## 알려진 한계 (v1 허용)

- AlertMonitor 쿨다운 상태는 인메모리 — 서버 재시작 시 소실
- HealthWatchdog의 broker ping이 `get_balance()` 호출로 구현됨 (전용 ping 없음)
- Web Notification은 브라우저 권한 허용 시에만 동작 (Settings에서 권한 요청 버튼 미구현)
