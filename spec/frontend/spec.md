## 1. 목표

React SPA로 StockVision의 사용자 인터페이스를 구현한다.
**듀얼 연결**(클라우드 서버 + localhost)로 클라우드 데이터와 로컬 실행을 통합.

**핵심 역할: 순수 UI**
1. 사용자 인증 (클라우드 서버) → JWT를 localhost에 전달
2. 전략 규칙 관리 (클라우드 서버 CRUD → 저장 후 localhost sync)
3. 실시간 시세 + 체결 모니터링 (localhost WS)
4. 실행 로그 열람, 설정 변경

> 컨텍스트 fetch, 하트비트, 템플릿은 로컬 서버가 직접 통신.
> 프론트엔드의 브릿지 역할은 **JWT 전달 + 규칙 sync**로 축소.

## 5. 연결 구조 + JWT 전달

### 5.1 연결 구조

```
프론트엔드
├── 클라우드 서버 연결 (HTTPS)
│   ├── 인증 (JWT)
│   ├── 규칙 CRUD
│   └── 어드민
│
└── localhost 연결 (HTTP + WS)
    ├── JWT 전달 (로그인 직후)
    ├── 규칙 sync (저장 직후)
    ├── API Key 등록
    ├── 상태 조회
    ├── 로그 조회
    └── WS: 실시간 시세 + 체결
```

> 컨텍스트, 하트비트, 템플릿은 로컬 서버가 직접 클라우드 서버와 통신.

### 5.2 JWT 전달 + 규칙 sync

```typescript
// 로그인 성공 시 — JWT를 localhost에 전달
async function onLoginSuccess(jwt: string, refreshToken: string) {
  await localClient.setAuthToken(jwt, refreshToken);
  // 이후 로컬 서버가 자립적으로 클라우드 서버와 통신
}

// 규칙 변경 시 — 저장 후 localhost에 sync
async function onRuleSave(rule: Rule) {
  // 1. 클라우드 서버에 저장
  await apiClient.saveRule(rule);

  // 2. localhost에 즉시 sync
  await localClient.syncRules(await apiClient.getRules());
  // (브라우저 닫혀도 WS push로 로컬 서버가 자동 re-fetch)
}
```

### 5.3 신호등 구현

```typescript
interface TrafficLight {
  apiServer: "green" | "red";       // 프론트가 직접 health check
  kiwoomProxy: "green" | "red";     // localhost /api/status 조회
  kiwoomApi: "green" | "red";       // localhost가 보고하는 키움 토큰 상태
}

// 5초마다 갱신
useInterval(() => {
  apiClient.healthCheck().then(ok => setApiServer(ok ? "green" : "red"));
  localClient.getStatus().then(s => {
    setKiwoomProxy(s.server === "running" ? "green" : "red");
    setKiwoomApi(s.kiwoom_connected ? "green" : "red");
  });
}, 5000);
```

## 6. 수용 기준

### 6.1 인증

- [ ] 회원가입 → 이메일 인증 → 로그인 성공
- [ ] JWT 만료 시 Refresh Token 자동 갱신

### 6.2 대시보드

- [ ] localhost WS 연결 → 실시간 시세 표시
- [ ] 체결 발생 → 알림 즉시 표시
- [ ] 신호등 3개 정상 표시

### 6.3 전략 빌더

- [ ] 매수/매도 조건을 UI로 구성 가능
- [ ] AND/OR 연산자 선택 가능
- [ ] 저장 → 클라우드 서버 + localhost sync

### 6.4 JWT 전달 + 규칙 sync

- [ ] 로그인 후 → JWT + Refresh Token을 localhost에 전달
- [ ] 규칙 저장 후 → localhost에 즉시 sync
- [ ] localhost 미연결 시 → 경고 표시 (규칙은 API에 저장됨)

### 6.5 설정

- [ ] API Key 입력 → localhost에 저장 (서버 미전송)
- [ ] 모의투자/실거래 모드 전환
