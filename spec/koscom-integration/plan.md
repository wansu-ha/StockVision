# 코스콤 연동 구현 계획서 (koscom-integration)

> 작성일: 2026-03-04 | 상태: **보류** | 대상: Phase 4+

---

## 결정 사항

**현재 구현 대상 아님.** 로컬 단일 사용자 모델 전환으로 우선순위에서 제외.

| 보류 이유 | 내용 |
|---------|------|
| 사업자 등록 필요 | 코스콤 핀테크기업 등록은 사업자 등록이 전제 |
| 대체 가능 | 로컬 모델에서 키움 COM API로 계좌 조회 가능 → 코스콤 불필요 |
| 복잡도 대비 이점 없음 | OAuth 2.0 동의 플로우 + 갱신 구현 부담 |

## 재검토 조건

- 유료 SaaS 확장 시점
- 다중 증권사(KB, 삼성 등) 지원 필요 시
- 사업자 등록 완료 후
