# 프론트엔드 UX 개선 v2

## 목표

대시보드 사용성 개선. 핵심 정보 접근성 향상, 불필요한 depth 제거.

## 요구사항

### 계좌 카드 재배치

| ID | 요구사항 | 우선순위 |
|----|---------|---------|
| U1 | 장 상태 표시(장전/장중/장후)를 계좌 카드 우측 상단에 배치. 장중=초록, 장전/장후=회색 | P0 |
| U2 | 엔진 신호등을 계좌 카드 우측 하단으로 이동 (Header에서 제거) | P0 |
| U3 | 신호등 옆에 "전략 실행" / "중지" 토글 버튼 배치 | P0 |
| U4 | 신호등 색상: 회색=미연결, 노랑=브로커만 연결, 초록=자동매매 중 | P0 |

### 레이아웃

```
┌──────────────────────────────────────────────────┐
│ 10,000,000원 +0%                        🔘 장전  │
│ 주문가능 10,000,000 │ 보유 0종목 │ 키움 모의  🟡 [전략 실행] │
└──────────────────────────────────────────────────┘
```

### Header 변경

| ID | 요구사항 | 우선순위 |
|----|---------|---------|
| U5 | Header 로고 옆 신호등 제거 | P0 |

## 수용 기준

- [ ] 계좌 카드에 장 상태 + 엔진 신호등 + 전략 실행 버튼 표시
- [ ] Header 신호등 제거
- [ ] 전략 실행 버튼 클릭 → `POST /api/strategy/start` 호출 → 신호등 초록
- [ ] 중지 버튼 클릭 → `POST /api/strategy/stop` 호출 → 신호등 노랑 유지
- [ ] 모바일(같은 WiFi)에서 대시보드 정상 표시

## 범위

**포함**: 계좌 카드 레이아웃, Header 신호등 제거, 전략 실행 버튼

**미포함**: broker-auto-connect (별도 spec), 전략 목록 UI 개선 (추후)
