## 개요

StockVision의 이용약관(Terms of Service)을 작성했습니다. 이 문서는 서비스의 법적 포지션을 명확히 합니다.

---

## 작성 내용

### 파일 위치
- `docs/legal/terms-of-service.md`

### 주요 섹션

#### 1. 제1-2조: 서비스 정의
- 서비스의 성격: **시스템매매 자동화 도구**
- 투자자문/투자일임 아님을 명시
- API Key 정의 (서비스 키 vs 유저 키)

#### 2. 제3조: 서비스 성격 명시
- 규칙 기반 자동 실행
- 회사가 투자 판단을 내리지 않음
- AI/LLM은 참고용 데이터 제공 역할

#### 3. 제4조: 투자 손실 면책
- 사용자의 규칙에 의한 손실은 사용자 책임
- 과거 성과 ≠ 미래 성과
- AI/LLM 분석의 한계 명시

#### 4. 제5조: 시스템 오류 면책
- 소프트웨어 버그, 네트워크 장애 면책
- 클라우드 장애 시 로컬 독립 동작 설명
- 예정된 점검 사전 공지
- 불가항력 사유 면책

#### 5. 제6조: 키움 API 사용
- 사용자가 직접 키움 계좌 개설
- API Key는 서버에 저장 금지
- Windows Credential Manager 저장
- 키움 약관 준수 강제

#### 6. 제7조: 데이터 수집
- 수집 항목: 이메일, 닉네임, 규칙
- 미수집 항목: API Key, 거래 기록, 잔고
- 운영 통계는 익명화 (개인정보 아님)

#### 7. 제8-13조: 기타 조항
- 이용 제한 및 금지 행위
- 약관 변경 절차
- 책임의 한계
- 준거법 (대한민국, 서울중앙지방법원)
- 버전 관리
