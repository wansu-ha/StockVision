## 주요 특징

### 1. 기술 수준의 상세함

**코드 예시 포함**:
```python
# 클라우드: Key를 환경변수에서만 읽기
self.api_key = os.getenv('KIWOOM_SERVICE_KEY')

# 로컬: Windows Credential Manager에서 읽기
self.user_key = self.read_from_credential_manager('kiwoom_api_key')
```

**데이터 흐름도 포함**:
```
[사용자 PC] → Credential Manager (DPAPI)
           → 로컬 서버 (localhost:8001)
           → 키움 WebSocket (유저 Key)
           → 로컬 로그 (SQLite)
           ↯ 클라우드로 금융정보 전송 금지
```

### 2. 법적 용어와 쉬운 표현의 균형

**예시**:
```
법적 용어: "투자자문 서비스가 아님"
쉬운 표현: "StockVision은 특정 종목을 추천하지 않습니다"

법적 용어: "불가항력 사유"
쉬운 표현: "정전, 네트워크 두절, 정부 규제 등 예측 불가능한 사건"
```
