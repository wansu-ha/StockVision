## 작성 내용

### 파일 위치
- `docs/legal/privacy-policy.md`

### 주요 섹션

#### 1. 제1조: 개요
- 개인정보 최소화 원칙
- 로컬 저장 원칙

#### 2. 제2조: 수집 항목
- **필수 수집**: 이메일, 비밀번호(해시)
- **선택 수집**: 닉네임, 전략 규칙
- **자동 수집**: 운영 통계 (익명화)
- 각 항목별 목적, 보관 위치, 보관 기간 명시

#### 3. 제3조: 미수집 항목
- **구조적 미수집** 원칙 강조
- 금융 정보 미수집: API Key, 체결 거래, 잔고, 계좌번호
- 로컬 저장 위치 명시

#### 4. 제4조: 보안 조치
- **클라우드 서버**:
  - HTTPS/TLS 1.3
  - Argon2id 비밀번호 해싱 (§8 준수)
  - AES-256-GCM 암호화 저장 (§10 준수)
  - 접속 기록 6개월 보관 (§9 준수)
  - AWS KMS/Vault 키 관리

- **로컬 서버**:
  - NTFS 파일 권한
  - BitLocker 디스크 암호화
  - Windows Credential Manager (DPAPI)

- **백업 및 복구**:
  - 매일 자동 백업 (AWS S3)
  - RTO/RPO 1시간

#### 5. 제5조: 개인정보 이용
- 이용 목적별 분류
- 제3자 제공 금지 (법적 요구 제외)
- 커뮤니티 데이터는 사용자 자발적 선택

#### 6. 제6조: 개인정보 삭제
- **회원 탈퇴**: 즉시 비활성화 → 30일 후 자동 삭제
- **로컬 데이터**: 사용자 책임 (우리 서버에 없음)
- **특정 데이터 삭제**: 3일 내 처리
- 삭제 권리 및 절차 명시

#### 7. 제7조: 법적 근거
- **개인정보보호법 제29조**: 안전조치의무
- **안전성확보조치기준**:
  - §8 비밀번호: Argon2id ✅
  - §9 접속기록: 6개월 ✅
  - §10 민감정보: AES-256-GCM ✅
- **키움 약관**: 제3자 위임 금지, 시세 중계 금지

#### 8. 제8-14조: 기타
- 사용자 권리 (열람, 정정, 삭제, 이의 제기)
- 쿠키 및 로컬 스토리지
- 미성년자 이용 제한 (만 18세 이상)
- 해외 사용자 (대한민국 법률 적용)
- 담당자 연락처
- 버전 관리

## 주의사항

### 1. 기술 스택 검증
- Argon2id는 Python passlib에서 지원
- AES-256-GCM은 AWS RDS의 암호화 기능
- Windows Credential Manager는 DPAPI 기본 사용

### 2. 보관 기간 확인
- 접속 기록: 안전성확보조치기준 §9 (6개월 이상)
- 회원정보: 탈퇴 후 30일
- 로그: 최소 1년 (내부 정책)

### 3. 운영 통계의 법적 해석
- 익명 UUID만으로 개인 식별 불가
- 개인정보보호법상 "개인정보" 아님
- 따라서 수집 동의 불필요
