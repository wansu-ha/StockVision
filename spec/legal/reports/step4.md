### 주요 섹션

#### 1. 제3자 위임 금지 (키움 약관 제5조②)

**키움 약관 조항**:
> 사용자는 보안 및 책임성 확보를 위하여 자신의 API Key를 제3자에게 대여, 양도, 공개하지 아니합니다.

**우리의 준수 방식**:

| Key 종류 | 소유자 | 저장 위치 | 접근 가능자 | 용도 |
|---------|-------|---------|-----------|-----|
| 서비스 키 | StockVision | 클라우드 환경변수 | API 서버 | 시세 수집 |
| 유저 키 | 사용자 | PC Credential Manager | 로컬 서버 | 거래 실행 |

**아키텍처 다이어그램** 포함:
- 사용자 PC: Credential Manager (DPAPI 암호화)
- 로컬 서버: 키움 WebSocket/REST 호출
- 클라우드: 환경변수로 서비스 키 관리
- 통신 경로 명시 (Key 전송 안 함)

**제3자 위임 방지**:
- ✅ 서비스 키는 환경변수로만 관리
- ✅ 유저 키는 클라우드에 저장 안 함
- ✅ 로컬 서버가 사용자를 대신하여 주문 실행 (사용자 판단 아님)
- ✅ 매매 책임이 명확히 사용자에게 귀속

#### 2. 시세 중계 금지 (키움 약관 제5조③)

**키움 약관 조항**:
> 사용자는 서비스 API로 수신한 시세 정보를 제3자에게 판매, 배포, 제공하거나 허락 없이 2차 활용을 금지합니다.

**우리의 준수 방식**:

**경로 A: 내부 분석용 (서비스 Key)**
```
클라우드 서버
  ├─ 서비스 Key로 키움 API 호출
  ├─ 시세 원본 데이터 클라우드 DB 저장
  ├─ AI 분석 실행 (감정 점수, 기술적 지표)
  └─ 사용자에게 제공: 분석 결과만
      ├─ "감정 점수: 75점" ✅
      ├─ "기술적 신호: 강세" ✅
      └─ 시세 JSON 원본 ❌
```

**경로 B: 사용자 시세 (유저 Key)**
```
로컬 서버
  ├─ 유저 Key로 키움 API 호출
  ├─ 시세 로컬 메모리 캐시 (규칙 평가용)
  ├─ 거래 기록 로컬 로그
  └─ 클라우드로 전송 금지
      ├─ 가격 데이터 ❌
      ├─ 거래 기록 ❌
      └─ 성공/실패만 보고 ✅
```

**경로 C: 커뮤니티 (v2)**
```
사용자 자발적 공유
  → 익명화 처리
  → 공개 통계만
```

**확인 사항**:
- ✅ 시세 원본 사용자에게 제공 안 함
- ✅ 시세 데이터를 제3자에게 판매/배포 안 함
- ✅ 로컬 데이터를 클라우드로 전송 안 함
- ✅ 커뮤니티 데이터는 익명화됨

#### 3. 기술 명세 및 보안

**서비스 Key 사용 예시** (Python):
```python
class DataCollectionService:
    def __init__(self):
        self.api_key = os.getenv('KIWOOM_SERVICE_KEY')
        self.api_secret = os.getenv('KIWOOM_SERVICE_SECRET')
```

**유저 Key 사용 예시** (Python):
```python
class LocalKiwoomClient:
    def __init__(self):
        # Windows Credential Manager에서 읽기
        self.user_key = self.read_from_credential_manager('kiwoom_api_key')
        # 클라우드로 전송 안 함
```

**가격 재확인 로직**:
```python
def verify_price_before_order(self, ticker, expected_price):
    current_price = self.get_realtime_price(ticker)
    # 5% 이상 차이 → 주문 거부
    if abs(current_price - expected_price) > expected_price * 0.05:
        return False
    return True
```
