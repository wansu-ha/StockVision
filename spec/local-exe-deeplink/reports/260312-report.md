## 빌드 시 발견된 이슈 (수정 완료)

1. **uvicorn 모듈 문자열 import 실패**: PyInstaller exe에서 `uvicorn.run("local_server.main:app")` 불가.
   → `uvicorn.run(app, ...)` 앱 객체 직접 전달로 수정.
2. **pystray, Pillow 미설치**: 트레이 아이콘 의존성 누락.
   → `pip install pystray Pillow`로 해결.
