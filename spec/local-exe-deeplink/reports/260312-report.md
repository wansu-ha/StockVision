## 변경 파일 요약

### 신규
- `local_server/utils/mutex.py` — CreateMutexW 래퍼, Windows 외 no-op
- `local_server/utils/deeplink.py` — HKCU 레지스트리 프로토콜 등록/검증

### 수정
- `local_server/main.py` — configure_logging 파일 핸들러, __main__ 블록 (mutex/port/deeplink/argv), health에 app 필드
- `local_server/pyinstaller.spec` — onefile→onedir, numpy/pandas excludes 제거, hiddenimports 추가
- `frontend/src/components/BridgeInstaller.tsx` — app 검증, 딥링크 시작 버튼, sv_installed 플래그

## 미검증 항목 (수동 필요)

- `pyinstaller local_server/pyinstaller.spec` 빌드
- exe 더블클릭 → 트레이 + health 응답
- Named Mutex 동작 (두 번째 실행 시 종료)
- 브라우저에서 `stockvision://launch` → exe 시작
- 포트 점유 시 토스트 + 종료
- `~/.stockvision/logs/server.log` 생성
