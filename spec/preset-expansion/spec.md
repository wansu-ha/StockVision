# 프리셋 보강 + DSL 함수 확장 — 기능 명세서

> 작성일: 2026-03-29 | 상태: 초안

## 1. 목표

기존 프리셋 7개에 8개를 추가하여 다양한 장세/난이도를 커버하고, 이를 위해 부족한 내장 함수를 확장한다. 전략 상태 요약을 표시하여 사용자가 자신의 전략 구성을 한눈에 파악할 수 있게 한다.

### 해결하는 문제

- 횡보장/모멘텀 전략 커버리지 부족 (기존 프리셋은 추세/청산 편중)
- 초보자용 프리셋 부족 (기존 7개 중 초보 1개)
- MACD 히스토그램, 스토캐스틱, 다이버전스 등 핵심 지표 미구현
- 전략의 매수/매도/손절 구성이 한눈에 보이지 않음

## 2. 현재 상태 (코드 확인 결과)

### 이미 구현된 함수 (builtins.py)

| 함수 | 파서 등록 | 평가기 구현 |
|------|----------|-----------|
| MACD(단기?, 장기?) | ✅ | context callable |
| MACD_SIGNAL(단기?, 장기?, 시그널?) | ✅ | context callable |
| EMA(기간) | ✅ | context callable |
| 볼린저_상단(기간?, 배수?) | ✅ | context callable |
| 볼린저_하단(기간?, 배수?) | ✅ | context callable |
| 상향돌파(A, B) | ✅ | ✅ 전용 구현 |
| 하향돌파(A, B) | ✅ | ✅ 전용 구현 |
| 평균거래량(기간) | ✅ | context callable |

### 이미 구현된 패턴 함수

| 패턴 | 정의 |
|------|------|
| 골든크로스 | 상향돌파(MA(5), MA(20)) |
| 데드크로스 | 하향돌파(MA(5), MA(20)) |
| MACD골든크로스 | 상향돌파(MACD(), MACD_SIGNAL()) |
| MACD데드크로스 | 하향돌파(MACD(), MACD_SIGNAL()) |
| 볼린저하단돌파 | 현재가 <= 볼린저_하단(20) |
| 볼린저상단돌파 | 현재가 >= 볼린저_상단(20) |
| RSI과매도 | RSI(14) <= 30 |
| RSI과매수 | RSI(14) >= 70 |

### 추가 필요

| 항목 | 유형 | 필요한 전략 |
|------|------|-----------|
| MACD_HIST(단기?, 장기?, 시그널?) | 함수 | #4 다이버전스 |
| STOCH_K(K기간?, 슬로잉?) | 함수 | #3 스토캐스틱 |
| STOCH_D(K기간?, 슬로잉?, D기간?) | 함수 | #3 스토캐스틱 |
| 강세다이버전스(지표, 기간?) | 함수 | #4 다이버전스 |
| 약세다이버전스(지표, 기간?) | 함수 | #4 다이버전스 |
| 등락률 | 필드 | #7 모멘텀 |

### 평가기 연결 확인 필요

파서(builtins.py)에 등록되어 있지만, 평가기(evaluator.py)에서 context callable로 위임하는 함수들은 **로컬 서버 엔진에서 context에 실제 계산 함수를 주입**해야 동작한다. 이미 구현된 함수(MACD, EMA, 볼린저 등)의 엔진 연결 상태를 plan 단계에서 확인해야 한다.

## 7. 프리셋 상세

### 7.1 MACD 골든크로스 (추세, 초보)

```dsl
MACD골든크로스 AND RSI(14) >= 50 AND 보유수량 == 0 → 매수 100%
MACD데드크로스 AND 보유수량 > 0 → 매도 전량
수익률 <= -3 → 매도 전량
```

### 7.2 볼린저밴드 + RSI 역추세 (역추세, 초보~중급)

```dsl
볼린저하단돌파 AND RSI과매도 AND 보유수량 == 0 → 매수 100%
볼린저상단돌파 OR RSI과매수 → 매도 전량
수익률 <= -4 → 매도 전량
```

### 7.3 스토캐스틱 과매도 반등 (역추세, 중급)

```dsl
슬로잉 = 3
상향돌파(STOCH_K(5, 슬로잉), STOCH_D(5, 슬로잉)) AND STOCH_K(5, 슬로잉) <= 25 AND MA(20) > MA(60) AND 보유수량 == 0 → 매수 100%
수익률 >= 4 OR STOCH_K(5, 슬로잉) >= 80 → 매도 전량
수익률 <= -3 → 매도 전량
```

### 7.4 MACD 히스토그램 다이버전스 (추세 전환, 고급)

```dsl
기간 = 20
강세다이버전스(MACD_HIST(), 기간) AND RSI(14) < 50 AND 보유수량 == 0 → 매수 100%
약세다이버전스(MACD_HIST(), 기간) AND 보유수량 > 0 → 매도 전량
수익률 <= -3 → 매도 전량
```

### 7.5 삼선 정배열 5/20/60 (추세, 초보)

```dsl
MA(5) > MA(20) AND MA(20) > MA(60) AND RSI(14) < 60 AND 보유수량 == 0 → 매수 100%
MA(5) < MA(20) AND 보유수량 > 0 → 매도 전량
수익률 <= -3 → 매도 전량
```

### 7.6 볼린저밴드 스퀴즈 돌파 (돌파, 중급)

```dsl
배수 = 1.5
상향돌파(현재가, 볼린저_상단(20)) AND 거래량 > 평균거래량(20) * 배수 AND 보유수량 == 0 → 매수 100%
고점 대비 <= -2 → 매도 전량
수익률 <= -3 → 매도 전량
```

### 7.7 장 초반 급등 모멘텀 (모멘텀, 중급)

```dsl
시간제한 = 30
목표수익 = 3
손절 = -2
등락률 >= 3 AND 장시작후 <= 시간제한 AND 거래량 > 평균거래량(20) * 3 AND 보유수량 == 0 → 매수 100%
수익률 >= 목표수익 → 매도 전량
수익률 <= 손절 → 매도 전량
장시작후 >= 180 AND 보유수량 > 0 → 매도 전량
```

### 7.8 EMA 크로스 추세 추종 (추세, 초보)

```dsl
단기 = 12
장기 = 26
상향돌파(EMA(단기), EMA(장기)) AND 보유수량 == 0 → 매수 100%
하향돌파(EMA(단기), EMA(장기)) AND 보유수량 > 0 → 매도 전량
수익률 <= -3 → 매도 전량
```

## 8. 다이버전스 알고리즘

### 강세다이버전스(지표, 기간)

```
1. 최근 N봉의 가격 데이터에서 로컬 저점 2개를 찾는다
   - 로컬 저점: 양쪽 봉보다 낮은 봉 (최소 2봉 간격)
2. 해당 시점의 지표 값을 구한다
3. 가격 저점2 < 가격 저점1 (가격은 더 내려갔는데)
   AND 지표 저점2 > 지표 저점1 (지표는 올라갔으면)
   → True (강세 다이버전스)
```

### 약세다이버전스(지표, 기간)

```
1. 최근 N봉의 가격 데이터에서 로컬 고점 2개를 찾는다
2. 해당 시점의 지표 값을 구한다
3. 가격 고점2 > 가격 고점1 (가격은 더 올라갔는데)
   AND 지표 고점2 < 지표 고점1 (지표는 내려갔으면)
   → True (약세 다이버전스)
```

구현 위치: `sv_core/parsing/evaluator.py` (상향돌파/하향돌파와 같은 패턴)
