# README 라이선스 섹션 초안

아래 내용은 공개 저장소의 `README.md`에 넣기 위한 초안입니다.
최종 반영 시 링크 경로와 공개 범위를 실제 배치에 맞게 조정해야 합니다.

```md
## 라이선스

이 저장소의 공개 코드는 [Mozilla Public License 2.0](https://www.mozilla.org/en-US/MPL/2.0/) (`MPL-2.0`)에 따라 배포됩니다.
최종 라이선스 본문은 저장소 루트의 `LICENSE` 파일을 기준으로 합니다.

### 왜 `MPL-2.0`인가

StockVision은 로컬에서 실행되는 엔진, 브로커 연동, 전략 평가 로직의 투명성이 중요한 프로젝트입니다.
우리는 사용자가 실행 구조와 안전장치를 직접 검토할 수 있도록 코드를 공개하되,
공개된 파일을 수정해 재배포하는 경우 그 수정 파일은 다시 공개되도록 `MPL-2.0`을 선택했습니다.

### 공개 범위

오픈소스로 제공되는 범위와 비공개로 유지되는 범위는 동일하지 않습니다.
자세한 범위는 [`OPEN_SOURCE_SCOPE.md`](docs/open-source/OPEN_SOURCE_SCOPE.md)를 참고해 주세요.

요약:

- 공개 대상: 로컬 실행 엔진, 브로커 어댑터, 전략 DSL/평가기, 로컬 대시보드, 공개 가능한 테스트/스키마
- 비공개 대상: 클라우드 운영 코드, 관리자 도구, 운영 자동화, 내부 연구 문서, 비밀정보, 재배포 권한이 정리되지 않은 자산

### 실행 파일과 소스 코드

실행 파일 또는 설치본을 배포하는 경우, 해당 릴리스에 대응하는 소스 코드 접근 경로를 함께 제공합니다.
공개 릴리스의 기준 소스는 이 저장소와 릴리스 노트에 명시된 태그 또는 커밋입니다.

### 상표

`MPL-2.0`은 코드에 대한 라이선스이며, 프로젝트 이름과 로고에 대한 상표권을 부여하지 않습니다.
`StockVision` 이름, 로고, 아이콘의 사용 규칙은 [`TRADEMARKS.md`](docs/open-source/TRADEMARKS.md)를 참고해 주세요.

### 면책

이 프로젝트는 투자 자문을 제공하지 않으며, 어떠한 수익도 보장하지 않습니다.
소프트웨어는 `as is`로 제공되며, 실거래 사용과 관련된 책임은 사용자에게 있습니다.
```
