## 3. 자본시장법 포지셔닝

### 3.1 핵심 질문: 투자일임업자인가?

**자본시장법 제6조 제7항**: 투자일임업 = "투자자로부터 금융투자상품에 대한 투자판단의 전부 또는 일부를 일임받아 투자자별로 구분하여 그 재산을 운용하는 것"

### 3.2 StockVision의 포지셔닝 근거

| 기준 | 투자일임업 | StockVision | 해당 여부 |
|------|---------|-----------|:--------:|
| 투자판단 일임 | 운용사가 판단 결정 | 사용자가 직접 규칙 정의 (조건, 수량, 종목) | ❌ |
| 재산 운용 | 운용사가 계좌 접근 | 사용자의 HTS에 COM API 연결, 사용자가 ID/PW 관리 | ❌ |
| 보수 수취 | 운용 보수 존재 | 소프트웨어 구독료 (투자 성과 연동 없음) | ❌ |
| 금융위 등록 | 필수 | 해당 없음 | ❌ |

**결론**: 사용자가 직접 작성한 규칙을 실행하는 **자동화 도구**에 해당. 투자일임업 미해당.

> 유사 사례: 증권사 HTS의 "자동매매" 기능, 키움증권 "영웅문 자동주문" — 모두 사용자 정의 조건 실행.

### 3.3 주의해야 할 표현
