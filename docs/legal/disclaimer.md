# StockVision 투자 위험 고지

**최종 갱신**: 2026-03-05
**버전**: 1.0
**효력 발생일**: 2026-03-05

---

## 주의: 투자 위험

**StockVision은 시스템매매 자동화 도구이며, 투자자문 또는 투자일임 서비스가 아닙니다.**

---

## 기본 면책 고지

### 1. 시스템매매의 특성

StockVision을 통한 투자는 다음과 같은 특징을 가집니다:

- **규칙 기반**: 사용자가 미리 정의한 규칙에 따라 자동으로 매매
- **판단 불개입**: 시스템이 매수/매도를 추천하지 않음
- **손실 위험**: 규칙이 손실을 만들 수 있음 (수익을 보장하지 않음)

### 2. 핵심 면책 문구

```
StockVision은 투자자문 서비스가 아닙니다.
StockVision은 투자일임 서비스가 아닙니다.

- 모든 매매 규칙은 사용자가 직접 정의합니다.
- AI/LLM이 제공하는 정보는 참고용 데이터이며, 투자 추천이 아닙니다.
- 과거 성과는 미래 수익을 보장하지 않습니다.
- 시스템 오류, 네트워크 장애로 인한 손실이 발생할 수 있습니다.
- 투자에 따른 모든 손익은 사용자 본인에게 귀속됩니다.

투자는 원금 손실의 위험이 있으며, 그 책임은 투자자 본인에게 있습니다.
```

## 제1조 투자 위험

### 1.1 시장 위험

주식 시장은 다음과 같은 요인으로 인해 급락할 수 있습니다:

- **경제 악화**: 경기 침체, 금리 인상, 인플레이션
- **정치 불안**: 정부 정책 변화, 규제 강화, 선거
- **기업 뉴스**: 실적 부진, 스캔들, 리더십 변경
- **산업 변화**: 기술 혁신, 경쟁 심화, 수요 감소
- **국제 이슈**: 환율 급변, 무역 전쟁, 지정학적 위험
- **자연재해**: 지진, 전염병, 재난

이러한 요인들은 **예측 불가능**하며, 과거 패턴이 반복된다는 보장이 없습니다.

### 1.2 유동성 위험

주식 시장의 거래 가능성(유동성)이 갑자기 악화될 수 있습니다:

- **거래량 급감**: 특정 종목의 거래가 중단되거나 매우 미흡할 수 있음
- **호가 폭주**: 매수/매도 호가의 차이가 크게 벌어질 수 있음
- **서킷브레이커**: 시장 급락 시 거래 중단 (KOSPI 지수 5% 하락 이상)
- **상한가/하한가**: 일일 변동폭 제한으로 원하는 가격에 매도 불가

이 경우 사용자의 주문이 원하는 가격에 체결되지 않을 수 있습니다.

### 1.3 시스템 오류 위험

StockVision의 소프트웨어나 하드웨어에서 오류가 발생할 수 있습니다:

- **버그**: 규칙 해석 오류, 거래 수량 계산 오류
- **네트워크 장애**: 인터넷 끊김, 라우터 오류
- **서버 장애**: 클라우드 서버 다운, 데이터베이스 오류
- **키움 API 장애**: 증권사 시스템 불안정, 주문 거부
- **PC 장애**: 사용자의 개인용 컴퓨터 오류, 전원 차단

이러한 오류로 인해 **손실이 발생할 수 있으며, 회사는 이에 책임지지 않습니다.**

### 1.4 타이밍 위험

자동 매매의 타이밍이 항상 최적이지 않을 수 있습니다:

- **주문 지연**: 규칙 판단에서 주문 실행까지 시간 차이
- **갭 다운**: 장 마감 후 호재/악재 뉴스로 다음날 급등/급락
- **스프레드**: 매수 호가와 매도 호가의 차이 (거래 비용)
- **슬리피지**: 주문한 가격과 실제 체결 가격의 차이

이러한 타이밍 손실은 매매 성과에 크게 영향을 미칠 수 있습니다.

### 1.5 규칙 설계 위험

사용자가 정의한 규칙에 오류가 있을 수 있습니다:

- **논리 오류**: 규칙이 의도와 다르게 작동
- **과적합(Overfitting)**: 과거 데이터에만 잘 맞는 규칙
- **변수 부족**: 중요한 조건을 놓친 규칙
- **극단적 손실**: 드물지만 거대한 손실이 발생할 수 있는 규칙 설계

예: "매일 주가 상승 시 최대한 많은 수량 매수" → 대폭락 시 거대 손실

## 제2조 과거 성과

### 2.1 과거 성과 ≠ 미래 성과

StockVision이 제공하는 백테스팅, 수익률 분석, 성과 차트 등은:

- **역사적 데이터 기반**: 과거의 시장 조건에서 규칙이 어떻게 작동했는지 보여줌
- **미래 성과 보장 안 함**: 동일한 조건이 반복된다는 보장 없음
- **시장 구조 변화**: 시장 참가자, 거래량, 변동성이 계속 변함
- **블랙스완(Black Swan)**: 예측 불가능한 극단적 사건 발생 가능

### 2.2 백테스팅의 한계

백테스팅(과거 데이터 테스트)은 다음과 같은 제약이 있습니다:

- **선택 편향(Survivorship Bias)**: 상장 폐지된 종목은 제외
- **비용 무시**: 거래 수수료, 세금, 슬리피지를 완전히 반영하지 않을 수 있음
- **극단적 상황 누락**: 2008년 금융위기, 2020년 팬데믹 같은 극단적 상황이 드묾
- **과적합**: 의도하지 않게 과거에만 최적화된 규칙

### 2.3 성과 표시의 주의점

- **총수익률**: 낮은 총수익률이라도 변동성이 매우 높을 수 있음
- **연간 수익률**: 특정 연도의 좋은 성과가 다른 연도에 반복되지 않을 수 있음
- **최대 낙폭(Drawdown)**: 규칙이 얼마나 손실을 감수하는지 나타냄
  - 만약 연 30% 수익이지만 최대 낙폭이 60%라면, 수익이 날아갈 위험 있음

## 제3조 AI/LLM 분석의 한계

### 3.1 AI 분석은 투자 추천이 아님

StockVision이 제공하는 AI 분석(감성 점수, 뉴스 요약, 데이터 인사이트):

- **참고 정보**: 투자 의사결정을 돕는 보조 정보일 뿐
- **투자 추천 아님**: "이 종목을 사세요" 같은 명령이 아님
- **완전성 보장 안 함**: 모든 중요 정보를 제공하지 않을 수 있음
- **오류 가능성**: AI는 실수할 수 있으며, 학습 데이터 편향 존재

### 3.2 AI의 한계

- **감성 분석 부정확**: 뉴스의 감정(긍정/부정)을 완벽히 판단하지 못함
- **뉴스 요약 누락**: 핵심 정보를 놓칠 수 있음
- **시장 심리 예측 불가**: AI는 인간의 비리성적 행동을 완벽히 예측 불가
- **시장 구조 변화 대응 지연**: 새로운 시장 패턴에 적응하는 데 시간 소요

### 3.3 AI에 의존하는 위험

AI 분석에만 의존하여 투자 결정을 내리면:

- 규칙을 정교하게 검증하지 않은 채 실행
- 감정적 오버스케일링 (AI 점수가 높다고 과도한 투자)
- 시스템 오류 시 손실

## 제4조 비용 및 수수료

### 4.1 거래 수수료

주식 거래 시 다음의 비용이 발생합니다:

| 수수료 종류 | 발생 주체 | 비용 |
|-----------|---------|------|
| 증권 수수료 | 증권사(키움증권) | 거래액의 약 0.015% (증권사 정책) |
| 거래세 | 정부 | 매도 시 거래액의 약 0.25% |
| 유지 비용 | 증권거래소 | 거래액의 약 0.00015% |

**예**: 100만 원 매수 후 100만 원에 매도 시
- 수수료: 150원 + 2,500원 = 약 2,650원 손실
- 실제로는 거래액의 약 0.27% 비용 발생

### 4.2 StockVision 비용 (향후 추가)

현재는 무료이나, 향후 다음과 같은 비용 모델이 추가될 수 있습니다:

- 구독료 (월 $9.99 예상)
- 프리미엄 AI 분석 (월 추가 비용)
- API 사용료 (고빈도 거래 시)

**비용은 항상 투명하게 공지됩니다.**

## 제5조 원금 손실 위험

### 5.1 100% 손실 가능성

극단적인 경우, 투자 원금을 **100% 잃을 수 있습니다**:

- **기업 파산**: 상장 폐지 회사에 투자한 경우
- **거대 손실**: 극악의 규칙 설계로 인한 연쇄 손실
- **시스템 오류**: 버그로 인한 대량 손실 주문 실행

### 5.2 손실의 심각성

손실은 다음과 같은 영향을 미칩니다:

- **심리적 트라우마**: 거대 손실로 인한 정신적 고통
- **재정 악화**: 잃은 원금을 복구하려다 추가 손실
- **빚 위험**: 증권사에서 신용거래를 한 경우, 손실이 빚으로 전환

### 5.3 신용거래의 위험 (레버리지)

신용거래(빌려서 투자)를 하면:

- **손실 확대**: 2배 투자 → 손실도 2배 이상
- **강제 청산**: 자산이 일정 수준 이하로 떨어지면 강제 매도
- **신용 이자**: 매일 이자가 발생하여 수익성 악화

**신용거래는 극히 주의하세요. 초보자는 절대 추천하지 않습니다.**

## 제6조 법적 책임

### 6.1 투자 손실은 사용자 책임

StockVision 및 회사는 다음에 대해 **책임지지 않습니다**:

- 규칙으로 인한 손실
- 시스템 오류로 인한 손실
- 네트워크 장애로 인한 손실
- 키움 API 장애로 인한 손실
- 시장 급락으로 인한 손실

### 6.2 법적 근거

이 면책은 다음을 기반으로 합니다:

- **증권거래법 제27조의2**: 투자자 스스로 투자 책임
- **금융감독 가이드라인**: 시스템매매 도구는 투자자문이 아님
- **키움증권 약관**: API 사용자가 거래 손실 책임

### 6.3 회사의 책임 한계

회사는 다음의 경우 책임을 질 수 있습니다:

- **의도적 기만**: 회사가 의도적으로 거짓 정보 제공
- **중과실**: 회사의 심각한 부주의 (예: 고의 보안 무시)

다만, 책임을 지더라도 배상액은 **사용자가 지난 3개월 이용료의 합계**를 초과하지 않습니다. (현재는 무료이므로 배상 불가)

## 제7조 주의사항 체크리스트

### 투자 전 확인하세요

- [ ] 원금 손실 위험을 이해했습니다
- [ ] 투자 결정은 전적으로 나의 책임임을 인정합니다
- [ ] StockVision은 투자 추천을 하지 않음을 이해했습니다
- [ ] 과거 성과가 미래를 보장하지 않음을 알고 있습니다
- [ ] 규칙 설계의 오류로 손실이 날 수 있음을 알고 있습니다
- [ ] 시스템 오류, 네트워크 장애 위험을 이해했습니다
- [ ] 신용거래(레버리지) 위험을 이해했습니다
- [ ] 여유 자금으로만 투자할 것입니다
- [ ] 정기적으로 규칙 성과를 검토할 것입니다
- [ ] 손실 발생 시 감정적으로 대응하지 않겠습니다

모든 항목에 동의한 후 투자를 시작하세요.

## 제8조 권장 사항

### 8.1 투자 시작 전

1. **여유 자금 준비**: 생활비를 제외한 여유 자금으로만 투자
2. **공부**: 주식 시장, 차트 분석, 시스템매매 개념 학습
3. **소액 시작**: 처음에는 최소 주문 단위(1주)로 테스트
4. **규칙 검증**: 백테스팅으로 규칙 검증 후 거액 투자

### 8.2 투자 중

1. **정기 검토**: 매주 또는 매월 규칙 성과 확인
2. **손실 관리**: 손실이 전체 자산의 5% 이상이면 규칙 일시 중지 검토
3. **다각화**: 여러 규칙과 종목에 분산 투자
4. **감정 제어**: 손실 후 즉흥적 거래 금지

### 8.3 투자 중단 신호

다음의 경우 투자를 중단하세요:

- 손실이 누적되어 원금의 20% 이상 손실 시
- 규칙이 계속 실패하고 개선 방법이 없을 때
- 시스템 오류가 반복될 때
- 정신적/신체적 스트레스가 심할 때

## 제10조 고지 동의

이 고지에 동의하지 않으면 StockVision 서비스 이용이 제한됩니다:

- ❌ 전략 활성화 불가
- ❌ 자동 매매 실행 불가
- ✅ 규칙 정의, 백테스팅은 가능

**동의 방식**:
- 회원가입 시: "투자 위험 고지에 동의합니다" 체크
- 전략 활성화 시: "위험을 이해하고 동의합니다" 팝업 확인
