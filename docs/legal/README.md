### 2. 개인정보처리방침 (Privacy Policy)

📄 `privacy-policy.md` (15KB)

**핵심 내용**:

**대상 사용자**: 모든 가입자

**동의 시점**: 회원가입 시 (필수)

### 3. 투자 면책 고지 (Investment Disclaimer)

**핵심 내용**:

- 비용 및 수수료
- 원금 손실 위험

**금감원 기준**:
- 과거 성과 고지

**대상 사용자**: 모든 투자자

**동의 시점**:
- 회원가입 시 (필수)
- 전략 활성화 시 (필수)

```
1. 회원가입 페이지
   ├─ 이용약관 동의 (필수) ← terms-of-service.md 링크
   ├─ 개인정보처리방침 동의 (필수) ← privacy-policy.md 링크

```

### 전략 활성화 플로우

```
1. 규칙 생성 → 저장
2. 전략 활성화 클릭
3. 투자 면책 고지 팝업 (필수) ← disclaimer.md 내용
   └─ "위험을 이해하고 동의합니다" 체크박스
4. 활성화 확인
```

## FAQ

### Q1: 이 약관은 법적 효력이 있나요?

**A**: 예, 있습니다. 다만 법원에서 최종 인정할지는 별개입니다. 따라서:
- 변호사 검수를 권장합니다
- 정기적으로 법무팀과 협의합니다
- 판례 변화에 따라 업데이트합니다

### Q2: 약관을 수정할 수 있나요?

**A**: 예, 가능합니다. 하지만:
- 변호사 검수 후 수정합니다
- 버전을 새로 부여합니다
- 기존 사용자에게 재동의를 요청할 수 있습니다
- 변경 이력을 보존합니다

### Q3: 사용자에게 정말로 이 약관을 강제해야 하나요?

**A**: 네, 필수적으로 동의받아야 합니다. 이유:
1. 법적 보호: 분쟁 시 증거 자료
2. 사용자 보호: 위험성을 명확히 알림
3. 규제 대응: 금감원, 개인정보보호위원회 검사 시 준비

### Q4: "투자 면책 고지"와 "투자자문 금지"의 차이는?

**A**:
- **투자자문 금지** → 법적 제한 (우리가 할 수 없음)
- **투자 면책 고지** → 사용자에게 위험을 알림 (우리가 해야 함)

약관은 후자에 포함되어 전자를 강화합니다.

### Q5: 로컬 데이터(거래 기록)는 약관에 영향을 받나요?

**A**: 아니요. 우리는 로컬 데이터를 수집하지 않으므로:
- 개인정보처리방침 대상 아님
- 약관에서 "미수집"으로 명시
- 사용자가 직접 관리 (우리 책임 아님)

### Q6: 약관은 언제 갱신되나요?

**A**: 다음 경우에 갱신됩니다:
- 법적 환경 변화 (법률 개정)
- 키움 약관 변경
- 서비스 기능 추가 (v2)
- 금감원 지침 변화

분기별로 검토합니다.
