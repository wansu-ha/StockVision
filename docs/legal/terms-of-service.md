## 제1조 목적

이 약관은 StockVision(이하 "서비스" 또는 "회사")이 제공하는 시스템매매 자동화 도구 이용에 관한 조건과 책임을 규정하기 위하여 제정되었습니다.

---

## 제2조 정의

### 2.1 서비스의 성격
- **StockVision**: 사용자가 직접 정의한 매매 규칙을 사용자의 개인용 컴퓨터에서 자동으로 실행하는 시스템매매 자동화 도구입니다.
- **서비스 범위**: 규칙 저장, 동기화, 실시간 시세 제공, 매매 기록 조회, AI 기반 데이터 제공(감성 점수, 뉴스 요약 등)

### 2.2 서비스의 한계
- StockVision은 **투자자문 서비스가 아닙니다**. 즉, 회사가 특정 종목 매수/매도를 추천하지 않습니다.
- StockVision은 **투자일임 서비스가 아닙니다**. 즉, 회사가 사용자를 대신하여 투자 판단을 내리지 않습니다.
- StockVision은 **주식 거래 중개 서비스가 아닙니다**. 회사는 증권사 API를 통해 사용자의 주문을 중개하지 않으며, 모든 거래는 사용자와 증권사 간의 직접 거래입니다.

### 2.3 규칙(Rule) 및 전략(Strategy)
- **규칙**: 특정 조건(가격, 거래량, 시간 등)을 만족할 때 자동으로 매수/매도를 실행하는 프로그램 로직입니다.
- **전략**: 하나 이상의 규칙을 조합한 집합입니다.
- 모든 규칙은 **사용자가 직접 정의**하며, 회사는 규칙의 유효성이나 수익성을 검증하지 않습니다.

### 2.4 API Key
- **API Key**: 증권사(키움증권 OpenAPI+, 한국투자증권 Open Trading API 등)에서 사용자에게 발급한 인증 정보입니다.
- **서비스 키**: 회사가 소유하고 시세 수집 목적으로만 사용하는 키입니다.
- **유저 키**: 사용자가 소유하고 자신의 계좌에서만 사용하는 키입니다.

## 제3조 서비스 성격 명시

### 3.1 시스템매매 도구
StockVision은 **시스템매매 자동화 도구**입니다. 사용자가 정의한 규칙에 따라 자동으로 매매를 실행하되:

- 모든 매매 판단과 책임은 **사용자에게 귀속**됩니다.
- 회사는 규칙 정의, 실행, 결과에 대해 책임지지 않습니다.
- 회사가 제공하는 AI/LLM 분석(감성 점수, 뉴스 요약 등)은 **참고용 데이터**일 뿐 투자 추천이 아닙니다.

### 3.2 금지된 사항
회사는 다음을 수행하지 않으며, 사용자도 다음을 요구할 수 없습니다:

- 특정 종목에 대한 매수/매도 추천
- 수익을 보장하거나 예측하는 분석
- 사용자 대신 투자 판단을 내리는 행위
- 법적 책임을 회사에게 전가하는 모든 형태의 투자자문

---

## 제4조 투자 손실 면책

### 4.1 기본 면책
사용자가 정의한 규칙에 의한 매매 결과로 인한 **손익은 사용자 본인에게만 귀속**됩니다. 회사는 다음에 대해 책임지지 않습니다:

- 사용자가 정의한 규칙의 유효성이나 수익성
- 규칙 실행으로 인한 손실
- 과거 성과와 무관한 미래 손실
- 매매 기회 상실로 인한 간접적 손실

### 4.2 과거 성과
- 과거의 수익률이나 성과는 **미래의 수익을 보장하지 않습니다**.
- 역사적 데이터를 이용한 백테스팅 결과도 실제 거래 성과를 보장하지 않습니다.
- 시장 상황, 유동성, 정책 변화 등에 따라 결과가 크게 달라질 수 있습니다.

### 4.3 AI/LLM 분석의 한계
회사가 제공하는 AI 기반 분석(감성 점수, 뉴스 요약, 데이터 인사이트 등)은:

- 참고용 정보일 뿐이며, 투자 추천이 아닙니다.
- 완전성, 정확성, 시의성을 보장하지 않습니다.
- AI의 오류나 학습 데이터 편향으로 인한 손실에 대해 회사는 책임지지 않습니다.

## 제5조 시스템 오류 및 장애 면책

### 5.1 시스템 오류
회사는 서비스 제공 과정에서 발생할 수 있는 다음 상황에 대해 **제한적으로 면책**됩니다:

- 소프트웨어 버그로 인한 오주문(오류 금액, 오류 수량)
- 네트워크 지연이나 끊김으로 인한 주문 실패
- 증권사 API 오류로 인한 주문 미체결
- 구식 시세 데이터(지연된 가격)를 기반으로 한 주문

### 5.2 시스템 장애로 인한 손실
회사는 다음과 같은 상황에서의 손실에 대해 책임지지 않습니다:

- 클라우드 서버 장애(다운타임, 느린 응답)
- 데이터베이스 오류
- 보안 침해 또는 해킹 시도
- 네트워크 불안정(사용자 ISP 또는 증권사 네트워크)

### 5.3 로컬 서버 독립 동작
로컬 서버는 클라우드 서버와 무관하게 독립적으로 동작할 수 있습니다:

- 클라우드 장애 발생 시, 로컬에 캐시된 규칙에 따라 매매 계속 실행
- 로컬 서버가 정상이면, 프론트엔드 접속 불가와 무관하게 규칙 실행 유지
- 클라우드 복구 후 규칙과 로그는 자동 동기화

### 5.4 예정된 점검
회사는 시스템 유지보수를 위해 **사전 공지와 함께 서비스를 일시 중단**할 수 있습니다:

- 점검 일시, 예상 소요 시간을 최소 48시간 전에 공지
- 점검 중 새로운 규칙 저장, 수정, 삭제 불가 (기존 규칙 실행은 유지)
- 점검으로 인한 거래 기회 상실에 대해 회사는 책임지지 않음

### 5.5 불가항력 사유
회사는 다음과 같은 불가항력 사유로 인한 서비스 중단에 대해 면책됩니다:

- 자연재해(지진, 홍수, 천둥번개 등)
- 정전 또는 통신 두절
- 정부 규제 또는 법원 명령
- 증권사 시스템 장애
- 전염병 확산, 전쟁 등 예측 불가능한 사건

## 제6조 증권사 API 이용

### 6.1 사용자 책임
- 사용자는 증권사(키움증권, 한국투자증권 등)에 직접 계좌를 개설하고 API 서비스를 신청해야 합니다.
- 사용자는 자신의 API Key와 Secret을 **절대로 서버에 전송하지 않습니다**.
- 사용자는 자신의 API Key 관리에 전적으로 책임지며, 유출 시 증권사에 즉시 신고해야 합니다.

### 6.2 회사의 책임 제한
- 회사는 사용자의 API Key를 **서버에 저장하지 않습니다**. (사용자 PC의 Windows Credential Manager에만 저장)
- 회사는 증권사 API의 오류, 변경, 중단에 대해 책임지지 않습니다.
- 회사는 각 증권사의 API 이용약관을 준수하며, 사용자도 이를 준수해야 합니다.

### 6.3 증권사 약관 준수
- 사용자가 StockVision을 통해 증권사 API를 사용할 때, 해당 증권사 API 이용약관의 모든 조항을 준수해야 합니다.
- 특히 다음을 엄격히 금지합니다:
  - 제3자에게 자신의 API Key를 대여 또는 양도
  - 증권사의 시세를 제3자에게 판매하거나 재배포
  - 약관에서 허용하지 않는 방식의 시세 수집

### 6.4 증권사 계약 해제
- 사용자가 증권사와의 계약을 해제하거나 API 권한을 취소하면, 해당 증권사 연동 기능은 작동하지 않습니다.
- 회사는 이를 사전에 감지하거나 알림하지 않으며, 사용자 책임입니다.

## 제7조 데이터 수집 및 활용

### 7.1 수집 데이터
회사는 다음 정보를 수집합니다:

- **계정 정보**: 이메일, 닉네임(선택), 비밀번호 해시
- **설정 정보**: 규칙, 포트폴리오 설정, 선호도
- **운영 통계**: 익명화된 앱 버전, OS, 키움 연결 상태, 타임스탐프 (개인정보 아님)

### 7.2 미수집 데이터
회사는 다음을 **절대로 수집하지 않습니다**:

- 사용자의 API Key, Secret
- 체결 내역, 잔고, 거래 기록
- 계좌번호
- 개인 신원(주민등록번호, 전화번호 등)

### 7.3 커뮤니티 데이터 활용(향후 기능)
v2 이상에서 사용자가 **자발적으로 공유한** 전략이나 수익 통계는:

- 서비스 개선, 통계, AI 학습 목적으로 활용될 수 있습니다.
- 공유를 철회하면 합리적 기간(최대 30일) 내에 모든 데이터가 삭제됩니다.
- 커뮤니티 공유 데이터는 익명화되어 처리됩니다.

## 제8조 서비스 이용 제한

### 8.1 금지 행위
사용자는 다음 행위를 해서는 안 됩니다:

- 불법 거래 또는 규제 회피 목적의 사용
- 회사 시스템에 대한 해킹, DDoS 공격, 또는 무단 접근
- 역공학, 소스 코드 추출, 복제
- 서비스 약관 및 법규 위반 사항을 반복적으로 수행하는 행위

### 8.2 이용 제한
위반 행위 적발 시 회사는 다음 절차에 따라 조치합니다:

1. **경고**: 위반 사실 통지 및 시정 요구 (이메일 또는 서비스 내 알림)
2. **일시 정지**: 경고 후 시정되지 않거나 중대한 위반인 경우, 사전 통지 후 계정 일시 정지
3. **영구 탈퇴**: 반복적 위반 또는 불법 행위의 경우, 영구 탈퇴 처리

다만, 해킹·DDoS 등 **긴급한 위해 행위**의 경우 즉시 정지 후 사후 통지할 수 있습니다.

## 제9조 서비스 수정 및 변경

### 9.1 수정 권리
회사는 운영상 필요에 따라 서비스의 일부 기능을 **사전 공지와 함께** 수정하거나 중단할 수 있습니다.

### 9.2 약관 변경
- 약관을 변경할 때는 **변경 내용을 공시**하고 충분한 유예 기간을 제공합니다.
- 사용자는 변경된 약관에 동의하지 않을 경우 **회원 탈퇴**할 수 있습니다.
- 변경 후 계속 서비스를 이용하면 변경된 약관에 동의한 것으로 간주됩니다.

## 제10조 책임의 한계

### 10.1 손해 배상
회사는 회사의 귀책사유(회사의 의도적 행위 또는 중과실)로 인한 손해 외에는 책임지지 않습니다.

### 10.2 간접 손해
회사는 **간접적, 부수적, 결과적 손해**(영업 손실, 기회 손실, 사업 중단 등)에 대해 책임지지 않습니다.

### 10.3 배상액 한도
회사가 책임을 지는 경우에도, 배상액은 **사용자가 지난 3개월간 서비스 이용료의 합계**를 초과하지 않습니다. (유료 서비스 기준)

## 제11조 준거법 및 관할

### 11.1 준거법
이 약관은 **대한민국 법률**을 따릅니다.

### 11.2 관할 법원
회사와 사용자 간의 모든 분쟁은 **서울중앙지방법원**의 관할을 따릅니다.

## 제12조 기타

### 12.1 약관의 일부 무효
약관의 일부 조항이 무효로 판단되더라도, 나머지 조항은 유효합니다.

### 12.2 저작권
StockVision 서비스의 모든 콘텐츠(코드, UI, 이미지, 문서)의 저작권은 회사에 귀속됩니다. 사용자는 개인 사용 목적으로만 열람할 수 있습니다.

### 12.3 연락처
약관 관련 문의: support@stockvision.kr

**이용약관 동의:**

이 약관에 동의하지 않으면 StockVision 서비스를 이용할 수 없습니다.

본 이용약관은 StockVision의 법적 포지션을 명확히 합니다:
- ✅ **시스템매매 도구**
- ✅ **자동화된 규칙 실행**
- ✅ **사용자 책임**
- ❌ 투자자문 서비스 아님
- ❌ 투자일임 서비스 아님
- ❌ 손실 보장 아님

**주의**: 이 약관은 법적 자문이 아닙니다. 법적 문제가 발생할 경우 전문 변호사와 상담하십시오.
