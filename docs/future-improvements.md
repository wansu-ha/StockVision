# 🔭 StockVision 추후 개선 사항 로드맵

본 문서는 MVP 완료 이후의 기능/품질/운영 개선 항목을 우선순위별로 정리합니다. 각 항목은 간단 목적, 기대 효과, 완료 기준을 포함합니다.

## 🧭 우선순위 기준
- P1: 1~2개월 내 고효과·저비용 개선 (MVP 신뢰도/현실성 강화)
- P2: 3~6개월 범위의 정확도/수익 안정화 개선
- P3: 장기 과제(실시간화/자산 확장/고급 데이터)

---

## 🅿️ P1 (고효율, 단기)

### 1) 레짐(시장 국면) 탐지 및 전략 스위칭
- 목적: 상승/변동성/위기 레짐에 따라 진입/청산/사이징 규칙 차등 적용
- 기대 효과: 드로우다운 완화, 회전율 최적화
- 완료 기준: 일별 레짐 산출(예: VIX/ATR/MA 기울기 조합)과 백테스트 성과 개선(Sharpe↑, MDD↓)

### 2) 동적 거래비용 모델(슬리피지/수수료)
- 목적: 스프레드·ATR·거래대금 기반 슬리피지 추정, 비용 현실화
- 기대 효과: 백테스트-실행 괴리 축소, 보수적 성과 산정
- 완료 기준: 비용 민감도 테스트에서 PF·Sharpe 하락폭 허용 범위 내(문서화)

### 3) 평가 견고성 강화(워크-포워드/PBO 점검)
- 목적: 과최적화 방지, 외부 구간 일반화 성능 확보
- 기대 효과: 운용 신뢰도 제고
- 완료 기준: Walk-Forward 결과·홀드아웃 6개월 성과 보고서, PBO 지표 허용 범위 내

### 4) 리스크·사이징 고도화(변동성 타게팅)
- 목적: ATR/변동성 기준 포지션 크기 조절, 일손실/연속손실 가드레일 정량 튜닝
- 기대 효과: 변동성 국면에서도 손실 제한 일관성 향상
- 완료 기준: 적용 후 MDD·좌측 꼬리 리스크 감소 확인

### 5) 데이터 품질 규칙 강화(결측/갭/스플릿/상폐)
- 목적: 데이터 이벤트 처리 표준화 및 자동 로그/재시도
- 기대 효과: 신호/성과의 일관성 향상
- 완료 기준: 품질 리포트(결측률, 갭, 조정 이벤트) 자동 생성

---

## 🅿️ P2 (중기)

### 6) 트리플 배리어 라벨링 + 메타-레이블링
- 목적: 진입 신호 유효성 2단계 판정으로 허위 신호 축소
- 기대 효과: 승률·PF 개선
- 완료 기준: 메타-레이블 도입 후 승률/손익비 통계 유의 개선

### 7) 규칙+ML 앙상블(가중/스태킹)
- 목적: 규칙 기반·RF·(추후 LSTM) 결합으로 안정성 확보
- 기대 효과: 전략 분산, 성과 변동성 감소
- 완료 기준: 단일 모델 대비 Sharpe↑, 드로우다운↓

### 8) 포트폴리오 구성(ERC/리스크 패리티/HRP 비교)
- 목적: 다종목 전략의 리스크 균형화
- 기대 효과: 벤치마크 대비 안정적 초과수익 추구
- 완료 기준: ERC/HRP 중 하나 채택 및 운용 KPI 개선

### 9) 실험/데이터 버저닝(MLflow, DVC)
- 목적: 실험 재현, 데이터/모델 버전 추적
- 기대 효과: 추적 가능성, 회귀 분석 용이
- 완료 기준: MLflow 대시보드·DVC 스토리지 연동

---

## 🅿️ P3 (장기)

### 10) 실시간화 및 에이전트 운영
- 목적: WebSocket/Celery/Redis 도입, 장중 신호·체결 대응
- 기대 효과: 전략 적용 범위 확장(분봉/실시간)
- 완료 기준: 지연 예산 내 실시간 대시보드 및 자동 체결 시뮬레이션

### 11) 대체 데이터/감성 신호
- 목적: 뉴스/NLP, 소셜, 재무지표 T+지연 반영
- 기대 효과: 이벤트 구간 대응력 강화
- 완료 기준: 감성 신호 추가 후 이벤트 기간 성과 개선

### 12) 운용 제약/용량 모델링
- 목적: 턴오버/체결 용량/슬리피지 비선형 모델링
- 기대 효과: 실거래 이전 단계 현실성 제고
- 완료 기준: 용량 제약 하 성과 유지 검증 리포트
