## 1. 오픈소스 Python 퀀트 프레임워크

### QuantConnect (LEAN Engine)
- **사용자**: 퀀트 개발자, 전문 리테일 알고 트레이더, 소규모 펀드
- **전략 정의**: Python / C# 코딩 (LEAN 오픈소스 엔진)
- **차별점**: 클라우드 기반 + 로컬 실행 모두 가능. 주식·옵션·선물·외환·크립토 멀티에셋. Alpha Marketplace (전략 수익화). 367,000+ 커뮤니티 회원. 2025년 AI IDE 통합(Copilot/Cursor) 추가
- **가격**: 백테스팅 무료, 라이브 트레이딩 유료 구독
- **핵심 기능**: 리서치→백테스트→라이브 단일 코드베이스, 기관급 데이터 번들

### Zipline
- **사용자**: 학술 연구자, 구 Quantopian 사용자
- **전략 정의**: Python (`initialize()` / `handle_data()` 이벤트 드리븐)
- **차별점**: Quantopian 레거시 — US 주식 팩터 전략에 특화. Pipeline API로 유니버스 선택
- **가격**: 무료 오픈소스
- **핵심 기능**: 현실적인 슬리피지·수수료 시뮬레이션. 단, Python 3.5-3.6 시대 설계 → 2025년 설치 난이도 높음. 라이브 트레이딩 사실상 단종

### Backtrader
- **사용자**: Python 입문 퀀트, 스윙 트레이더
- **전략 정의**: Python (class 기반, Pythonic API)
- **차별점**: 가장 낮은 진입 장벽. IB/OANDA 직접 연결 라이브 트레이딩 지원. 자체 데이터 없음 (CSV 등 직접 공급)
- **가격**: 무료 오픈소스
- **핵심 기능**: 심플한 구조, 커뮤니티 문서 풍부. 대용량 데이터나 분봉 이하 인트라데이에서 속도 한계

### Jesse
- **사용자**: 크립토 퀀트 트레이더
- **전략 정의**: Python (클린 API, 멀티타임프레임 지원)
- **차별점**: **크립토 전용**. 룩어헤드 바이어스 제로 보장 설계. 완전 자가호스팅 (전략 IP 보호). Optuna 통합 파라미터 최적화. 내장 AI 어시스턴트
- **가격**: 프리미엄 / 원타임 구매 (self-hosted)
- **핵심 기능**: 백테스트 정확도가 가장 높음. 주식·선물 미지원

## 2. 데스크톱 전문 플랫폼

### Sierra Chart
- **사용자**: 선물·CME 전문 단기 트레이더, 프롭 트레이더
- **전략 정의**: (1) 스프레드시트 자동화 (쉬움), (2) ACSIL (C++ 수준 고성능 커스텀 스터디), (3) DTC/FIX 프로토콜 직접 연결
- **차별점**: 극도로 빠르고 안정적. 변동성 피크에서도 랙·크래시 없음. CME/CBOT/EUREX 직접 데이터. 동시 실행 자동매매 시스템 수 제한 없음. 10년 이상 클리어링 펌 신뢰
- **가격**: 월정액 구독 (티어별, 상당히 저렴한 편)
- **핵심 기능**: 전문 데이 트레이더가 가장 신뢰하는 인프라 수준 안정성. 학습 곡선 급격

### MultiCharts
- **사용자**: CTA, 재무 어드바이저, 선물·주식 시스템 트레이더
- **전략 정의**: EasyLanguage® 및 .NET (TradeStation 언어와 호환)
- **차별점**: EasyLanguage 호환으로 TradeStation 전략 재사용. Portfolio Trader 내장 (수백 종목 동시 백테스트 + 자동매매). 100+ 성과지표. 20+ 데이터피드, 10+ 브로커 연결. Walk-Forward Optimization 내장
- **가격**: 연간 라이선스 또는 일회성 구매 (수백 달러 수준)
- **핵심 기능**: TradeStation에서 넘어오는 트레이더들의 1순위 대안. 포트폴리오 레벨 자동매매가 강점

### TradeStation (EasyLanguage)
- **사용자**: 미국 주식·선물·옵션 액티브 트레이더, 초중급 알고 트레이더
- **전략 정의**: EasyLanguage — "Buy market", "Sell 100 shares" 같은 영어 유사 문법
- **차별점**: 1980년대 창업, 가장 오래된 리테일 알고 플랫폼. RadarScreen® 실시간 멀티심볼 스캐너. DMA 직접 주문 라우팅. 2025년 AI + EasyLanguage 조합으로 진입 장벽 급감
- **가격**: 브로커 계좌 기반 (잔고 조건 충족 시 플랫폼 무료)
- **핵심 기능**: 영어 유사 문법 → 비프로그래머 접근성. MultiCharts가 이 언어를 그대로 채택

### AmiBroker (AFL)
- **사용자**: 리테일 퀀트, 주식 시스템 트레이더 (특히 아시아)
- **전략 정의**: AFL (AmiBroker Formula Language) — 벡터 연산 기반 자체 언어
- **차별점**: **세계 최고 속도 포트폴리오 백테스팅** — 2GHz CPU에서 1억 6600만 바/초 처리. 멀티스레딩 32코어 지원. Particle Swarm + CMA-ES AI 최적화. 일회성 구매 모델 (저렴)
- **가격**: Standard $279 / Professional $339 / Ultimate Pro $499 (일회성, 24개월 업그레이드 포함)
- **핵심 기능**: 백테스팅 속도는 기관급 플랫폼(Deltix, QuantHouse) 능가. 라이브 자동매매는 IB API 외 연결 복잡

## 3. 메타트레이더 생태계

### MetaTrader 4 / 5 (MT4/MT5)
- **사용자**: 외환·CFD 리테일 트레이더 전 세계 수백만 명
- **전략 정의**: MQL4/MQL5 (자체 C++ 유사 언어), Expert Advisors (EA)
- **차별점**: 브로커 무료 제공 → 진입 비용 0. MQL5.community 700만 방문자/월. EA 마켓플레이스로 전략 구매·판매. MT5: 21 타임프레임, 38 내장 지표
- **가격**: 브로커 계좌 보유 시 무료
- **핵심 기능**: 가장 넓은 생태계. 외환/CFD에 편중, 한국 주식 미지원

## 4. 프로 데이 트레이더 실제 셋업

### 실행 플랫폼 (Execution)
- **Interactive Brokers TWS**: 135개국 시장, API 자동화, 기관급 주문라우팅 → 테크 사비 트레이더 1순위
- **thinkorswim (Schwab)**: 기술적 분석 최강. ThinkScript 자체 언어. 미국 옵션·주식 트레이더 표준
- **Sterling Trader Pro**: 프롭 펌·기관 스캘퍼 전용. 핫키 실행, 멀티계좌 관리
- **DAS Trader**: 수십 개 브로커 지원 → 브로커 바꿔도 UI 동일. 프롭 트레이더 표준
- **Lightspeed Trader**: 100개+ 주문 라우팅, 초저지연. 스캘퍼 특화

### 차팅·분석
- **TradingView**: 차팅·워치리스트 표준 (거의 모든 트레이더 보조 도구로 사용)
- **Trade Ideas**: AI 스캐너, Holly AI 자동 신호. 미국 주식 데이 트레이더 필수
- **Benzinga Pro**: 실시간 뉴스 스캐닝 1위

### 저널링
- **TraderSync**: 자동 성과 분석, 패턴 식별

## 5. 한국 시스템트레이딩 도구

### 영웅문 (키움증권)
- **사용자**: 국내 개인 투자자 최다 사용 HTS
- **전략 정의**: Signal Language (TradeStation EasyLanguage 호환 설계, MultiCharts Power Language 호환)
- **차별점**: 초급/중급/고급 3단계 전략 작성 지원. KIS API 연동. 20년+ 국내 리딩
- **가격**: 계좌 보유 시 무료
- **핵심 기능**: EasyLanguage 호환으로 해외 전략 이식 가능성

### 예스트레이더 (iM증권, 구 이베스트투자증권)
- **사용자**: 국내 시스템 트레이더, 코스피/코스닥 자동매매
- **전략 정의**: YesLanguage (자체 언어) + JavaScript Language 지원
- **차별점**: 거래소·코스닥 전 종목 종목검색식 지원. 장중 조건 검색 + 자동매매 통합
- **가격**: 계좌 보유 시 무료
- **핵심 기능**: JavaScript 지원으로 프로그래머 친화적

### 사이보스 (CYBOS, 대신증권)
- **사용자**: 국내 고급 시스템 트레이더
- **전략 정의**: COM 기반 API (Python/C++ 연동)
- **차별점**: 개인 투자자 수준에서 가장 낮은 레이턴시 API 중 하나로 알려짐
- **가격**: 계좌 보유 시 무료
- **핵심 기능**: COM API → Python으로 직접 제어. 슈퍼개미들이 파이썬 자동매매에 활용

### 국내 트렌드 (2015~현재)
- 슈퍼개미들이 Python으로 HTS를 커스텀 개조하는 사례 급증
- KIS/키움 REST API + Python 조합이 리테일 알고 표준화
