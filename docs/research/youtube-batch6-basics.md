### 영상 2: 개미 투자자가 유리한 이유? 개미들의 핵심 주식 투자 전략!
- **채널**: 미키피디아
- **URL**: https://www.youtube.com/watch?v=D-0Wlfh_dok
- **길이**: 24:01
- **언어**: 한국어

#### 핵심 주제
개인 투자자(개미)가 기관 대비 가지는 6가지 구조적 장점과 실전 전략.

#### 개미의 6가지 구조적 장점

1. **장기 투자 자유**: 기관은 고객 단기 성과 압박 존재. 개미는 내 돈으로 내가 원하는 만큼 보유 가능
   - 발화자 보유 종목: 삼성전자 22년, 구글·S&P500 18년, 메타 14년

2. **빠른 의사결정**: 기관은 내부 보고·승인 프로세스 필요. 개미는 즉시 결정 가능
   - 단, 감정적 결정의 위험도 동반

3. **동등한 정보 접근**: 기업 실적 발표, CEO 인터뷰 모두 온라인 공개 — 기관과 동시에 접근 가능
   - 정보 접근보다 **해석 능력**이 핵심 경쟁력

4. **낮은 운영 비용**: 사무실·인건비 없이 스마트폰만으로 운영 가능

5. **자유로운 포트폴리오 구성**: 기관은 종목별·업종별 비중 제한 존재. 개미는 제한 없음

6. **지수 투자 가능**: 기관은 지수를 이기지 않으면 존재 이유 없음. 개미는 지수 그대로 따라가도 충분

#### 개미 핵심 전략

**액티브 인컴 + 패시브 인컴 병행 원칙:**
- 근로소득(Active Income)으로 투자 원금 확보
- 투자 수익(Passive Income)으로 자산 확대
- "Pay Yourself First" — 소비 전 투자 먼저

**시장 비이성성 활용:**
- 시장은 종종 외부 요인(케빈 워시 지명 루머, 딥시크 쇼크 등)으로 과도 반응
- 개미의 할 일: **패닉하지 않고** 펀더멘탈 좋은 기업을 할인된 가격에 매수
- 3가지 기준: ① 펀더멘탈 우수 ② 장기 포트폴리오에 추가하고 싶은 기업 ③ 외부 요인이 펀더멘탈에 실제 영향 없을 것

**분산 전략 ("둘 다 사라"):**
- 구글 vs 마이크로소프트 논쟁 → 3년 전 반반 매수 시 2.7배 수익 (구글만 3.5배, MS만 1.8배)
- 구글 vs MS의 분위기 전환은 예측 불가 → 둘 다 사는 것이 합리적
- 삼성전자 vs SK하이닉스 → 둘 다 사라

**포트폴리오 관리 원칙:**
- "꽃이 피면 꽃에 물을 줘라, 잡초에 물 주지 마라" → 오르는 주식을 팔지 말고 유지
- 많이 올랐다 = 고평가가 아니다 (실적이 같이 올랐다면)
- 많이 떨어졌다 = 저평가가 아니다 (실적과 무관하게 심리로 떨어졌을 수 있음)
- 지수 ETF가 개미에게 최고의 분산 수단 (자동 종목 클린징 효과)

**요트 투자 비유 (책 인용):**
- 뒤를 보며 노 젓기 = 과거 주가 차트 트레이딩 (잘못된 방법)
- 파도 타는 카약 = 시장 타이밍 맞추기 (어려운 방법)
- 바람 타는 요트 샴페인 마시기 = 장기 정립식 투자 (권장 방법)

#### StockVision 관련 시사점
- **리스크 경고 설계**: 기관 대비 개미의 약점(감정 결정, 정보 해석 부족)을 보완하는 AI 기능 필요
- **전략 다양성**: 단기 트레이딩뿐 아니라 장기 보유 전략도 지원해야 타겟 사용자 확장
- **포트폴리오 UI**: "둘 다 사라"류의 분산 추천 기능, 자동 리밸런싱 알림 기능 검토
- **패닉 방지 기능**: 시장 급락 시 자동매매 중단보다 전략 원칙 상기 알림 기능

ETF 투자 기초, 배당 투자로 월 300만원 수동 소득 달성, 투자 시작 시점 조언.

#### 월 300만원 배당을 위한 계산
- 배당 수익률 6% 가정: 연 3,600만원 = **원금 6억원** 필요
- 코카콜라 배당 수익률 3% → 워렌 버핏의 실제 배당 수익률 66% (1984년 매수 기준)
- **핵심**: 배당킹(50년 이상 배당 성장)은 금리·경기와 무관하게 지속 지급 — 비즈니스 안정성 증명

#### ETF 투자 기초

**ETF의 장점:**
1. **소액 분산 투자**: 16,000원으로 미국 500개 기업 보유 (S&P500 ETF)
2. **접근성**: 달러 환전, 밤샘 미국장 대기, 금·채권·원자재 등 복잡한 절차 없이 클릭 한번

**추천 ETF 구성 (3분의 1 전략):**

- 한국: KOSPI 200 또는 밸류업 ETF
- 이머징: 인도, 베트남 등 포스트 차이나

**SCHD (슈드) 추천 이유:**
- 미국 배당 성장주 ETF — 배당금이 매년 증가하는 기업 모음
- 장기 투자 시 매수가 대비 배당 수익률이 지속 상승
- 국내 접근법: ISA 계정에서 "미국 배당 다우존스" 검색 → 동일 상품 10여개 존재

**투자 시작 시점에 대한 조언:**
- "지금 사면 늦은 거는 맞다. 하지만 지금 시작한다면 괜찮다"
- 핵심: 목돈 일시 투자 반대. **정립식 장기 투자 + 비중 조절**
- 역사적으로 S&P500을 20년 이상 보유하면 손실 확률 0%
- "100 - 나이 = 주식 비중" 공식 소개

**손실 경험 공유:**
- 본인도 솔깃한 정보에 속아 연봉 수준 자금 전액 손실 경험
- 그 이후 100배 수익 제안에도 귀 닫게 됨
- 교훈: 투자는 "예체능" — 이론보다 실전, 손실 경험이 더 빠른 학습

### 영상 4: 투자 초보자 필수 영상. 이 영상은 당신의 인생을 바꿀겁니다.
- **채널**: 쉽알남
- **URL**: https://www.youtube.com/watch?v=-Tp2fhvVVGM
- **길이**: 14:20
- **언어**: 한국어

#### 핵심 주제
단타 트레이딩 입문. 기초 기술적 분석 → 리스크 관리 → 마인드셋 순서의 3단계 구조.

#### 단타 트레이딩 준비물
1. TradingView (차트 도구)
2. 거래소 계정
3. 시드 — 처음엔 소액 (주식: 월급 수준 200~300만원, 코인: 10만원)
4. 매매 일지 (간단하게, 1분 내 작성 가능한 양식)

> **시뮬레이션 거래 비추천 이유**: 뇌에 "가짜 돈"으로 각인되어 실력이 늘지 않음. 실제 돈(잃으면 약간 아픈 금액)으로 훈련해야 함.

#### 기초 기술적 분석: 오더블록(Order Block)

**원리:**
- 이전 캔들의 몸통을 완전히 감싸는 반대 방향 캔들 발생 시 → 감싸진 캔들 몸통 = 오더블록 구간
- 양봉 몸통 감싼 음봉 → 양봉 몸통 구간 = **저항 구간 (매도)**
- 음봉 몸통 감싼 양봉 → 음봉 몸통 구간 = **지지 구간 (매수)**

**진입·손절·익절 규칙:**
- **진입**: 오더블록 구간 진입
- **손절**: 오더블록 형성 캔들의 최저점·최고점 이탈 시
- **익절**: 고점 돌파 시 보유량 1/2 익절 → 나머지는 매수 평단가에 스탑로스 설정

#### 리스크 관리 3원칙

1. **손절 자리를 정하고 반드시 지켜라**
   - "퇴학당하는 트레이더의 90%는 손절 자리를 어기고 억지를 부리다 죽는다"
   - 머리로는 알지만 탐욕 때문에 못 지킨다

2. **반익 반본 (Half-profit, Break-even stop)**
   - 반익절 후 평단가에 스탑로스 → 분석이 틀려도 수익으로 마감 가능
   - "분석이 틀려도 수익을 얻는 것" — 이것이 핵심

3. **출금 습관**
   - 거래소 내 사이버머니 = 아직 수익이 아님
   - 규칙: 수익 발생 시 일부(10~20%) 무조건 출금
   - 원금 회수 후 수익금 전액 출금하는 방식도 유효
   - "오래 살아남을수록 강한 시장" — 단기 고수익보다 장기 생존

#### 마인드셋

- 트레이딩 심리는 노력보다 **재능에 가까움** — 단, 훈련 가능
- 본인 트레이딩 원칙 14가지를 작성 후 매일 매매 전 읽는 루틴 확립
- 내동 매매(감정 매매) 발생 시마다 원칙 추가 → 14가지로 압축
- **기술적 분석은 외울 수 있지만, 리스크 관리·마인드셋은 실천이 핵심**

**이 시장에서 살아남는 법:**
- 손절 + 반익반본 + 출금 — 이 세 가지가 시드 보호의 전부
- "매매는 예술, 매도는 더 큰 예술" — 익절이 진입보다 어렵다

#### StockVision 관련 시사점
- **자동 손절 기능**: 단타 전략에서 손절 자리 자동 설정 및 강제 실행 기능 필수
- **반익절 자동화**: 목표가 도달 시 절반 자동 청산 + 나머지 본절 스탑로스 설정 기능
- **매매 일지 통합**: 플랫폼 내 매매 후 자동 로그 기록 기능
- **출금 알림**: 수익 달성 시 일부 인출 권장 알림 (행동 금융학 적용)

#### 핵심 주제
포트폴리오 이론 + 자산배분. "돈을 잃지 않는 것"이 투자의 시작이자 수학적으로 증명된 원리.

#### 구멍 막기 비유

신입사원 연수의 "구멍 난 통에 물 채우기" 게임:
- 물을 많이 붓는 팀보다 **구멍을 막는 팀**이 항상 1등
- 투자도 마찬가지: 수익을 많이 내는 것보다 **손실을 막는 것이 우선**
- 포트폴리오 이론의 핵심은 "수익률 최대화"가 아니라 **"마이너스 확률 최소화"**

#### 현대 포트폴리오 이론 설명

**기본 원리:**
- 단일 자산의 방향 예측은 불가능 (랜덤하게 움직임)
- 하지만 **상관관계가 낮은 자산을 섞으면** 손실 확률을 수학적으로 낮출 수 있음
- "전 세계 모든 기금이 이 이론을 차용 — 투자 이론의 알파요 오메가"

**예시 (주식 + 달러 조합):**
- 코스피 17% 하락 시 → 달러 20% 상승 (IMF, 리먼 사례 역사적으로 증명)
- 주식 70% + 달러 30% 포트폴리오 → 주식 하락 시 달러가 완충
- 비중이 틀어지면 리밸런싱 (주식 비중 낮아지면 달러 팔아 주식 추가 매수)
- 이 과정을 반복하면 **방향 예측 없이도 수익** 가능

**포트폴리오 이론의 트레이드오프:**
- 좋은 것: 하락을 방어함
- 아쉬운 것: 상승도 제한됨 (욕심을 버리면 하락을 막는다)
- "국민연금의 안정적인 수익률 = 포트폴리오 이론의 결과, 천재 운영자가 있어서가 아님"

**실용 조언:**
- 개인은 2~3개 자산(주식 + 달러/채권) 비중 조절만으로 손실 방어 충분
- 분기 1회 또는 시장 급락 때 리뷰
- 채권 추천: 금리 고점 구간에서 미국채 매력 — 30년물 5% 수익률 = 30년 후 4배
- 목표 수익률: 기준금리 + 2% (현재 한국 3.5% + 2% = 5.5%)

**한국 투자 환경 비판:**
- 코스피가 20여년 간 2,000~3,000에 갇힌 이유: 연금의 주식시장 연계 부족
- 미국은 401k 퇴직연금이 주식시장으로 자연 유입 → 자산시장 성장
- 전세 사기 피해의 본질: 타인의 투자 실패가 개인 자산 타임라인을 파괴
- "선진국이 된다는 것은 투자가 선택이 아닌 필수가 되는 것"

#### StockVision 관련 시사점
- **포트폴리오 리밸런싱 기능**: 자산 비중 모니터링 및 자동/수동 리밸런싱 알림
- **리스크 점수 표시**: 현재 포트폴리오의 상관관계 기반 리스크 수준 시각화
- **목표 수익률 설정**: 기준금리 연동 목표 수익률 설정 가이드
- **손실 방어 우선 UI**: "얼마를 벌었나"보다 "얼마를 지켰나" 지표 표시

## 크로스 컷 인사이트: StockVision 플랫폼 설계 함의

### 리스크 관리 기능

| 영상에서 나온 원칙 | StockVision 구현 방향 |
|---|---|
| 손절 자리 정하고 반드시 지켜라 (영상4) | 전략 설정 시 손절가 필수 입력, 자동 손절 실행 |
| 반익절 후 본절 스탑로스 (영상4) | 목표가 도달 시 분할 청산 + 평단가 손절 자동 전환 |
| 출금 습관 (영상4) | 수익 달성 시 일부 인출 권장 알림 기능 |
| 포트폴리오 상관관계 분산 (영상5) | 동일 섹터 집중 시 경고, 자산 다양성 점수 표시 |
| 패닉 매도 방지 (영상2) | 급락 시 전략 원칙 팝업, 자동매매 일시 중단 옵션 |

### 사용자 교육 콘텐츠

| 교육 주제 | 근거 영상 | 구현 제안 |
|---|---|---|
| 장기 투자 vs 단기 트레이딩 차이 | 영상1·2·3 | 온보딩 투자 스타일 설문 + 맞춤 전략 추천 |
| 월 N만원 수동소득을 위한 목표 원금 | 영상3 | 목표 설정 계산기 (배당 수익률 기반) |
| 분산투자의 수학적 원리 | 영상5 | 포트폴리오 리스크 점수 + 시뮬레이션 |
| 감정 매매 방지 원칙 | 영상4 | 매매 전 원칙 체크리스트 표시 |
| 시장 비이성성 활용 | 영상2 | 급락 감지 시 "저점 매수 기회" vs "공포" 분류 알림 |

### 자동매매 전략 설계 원칙

영상들이 제시한 원칙을 StockVision 전략 엔진에 적용:

```
전략 설계 필수 요소:
1. 진입 조건 (기술적 분석: 오더블록, 지지/저항)
2. 손절가 (필수, 비어있으면 전략 실행 불가)
3. 목표가 (분할 청산 비율 설정 가능)
4. 포지션 크기 (포트폴리오 대비 최대 비중 제한)
5. 매매 일지 자동 기록
```

### 자동화 역설 해소

영상 2의 "요트 투자" 비유와 자동매매 플랫폼의 관계:
- 단기 트레이딩 자동화 = 카약(파도 타기)에 해당할 수 있음
- 장기 원칙 기반 자동매매 = 요트(바람 타기)에 근접
- **StockVision의 포지셔닝**: 감정 없이 원칙을 실행하는 "시스템 레버리지" → 사용자의 시간을 자유롭게 하는 도구
