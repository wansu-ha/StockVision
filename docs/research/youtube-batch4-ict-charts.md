### 핵심 ICT/SMC 개념

#### 1. Smart Money vs Dumb Money 구분
- **Smart Money**: 기관/세력, 논리적 사고, 유동성 공급자
- **Dumb Money**: 개인/리테일, 본능적 반응(FOMO/공포), 유동성 소비자
- 전략적 함의: 기관의 행동 패턴을 읽어 역방향 트랩을 피함

#### 2. 유동성 (Liquidity)
- 정의: 돈이 쌓여 있는 위치 = 스탑로스가 몰린 곳
- 종류:
  - **Buy-Side Liquidity (BSL)**: 고점 위에 쌓인 매수 스탑 → 세력이 매도를 위해 스윕
  - **Sell-Side Liquidity (SSL)**: 저점 아래에 쌓인 매도 스탑 → 세력이 매수를 위해 스윕
- 최종 타겟: 유동성 위치를 타겟으로 설정
- **자동화 포인트**: 스윙 고점/저점을 감지하고 그 바깥 유동성 존을 마킹

#### 3. 효율성 (Efficiency) = FVG (Fair Value Gap, 공정 가치 갭)
- 정의: 급격한 가격 이동으로 생긴 미충전 구간 (3캔들 패턴)
- 형성: 1번 캔들 High와 3번 캔들 Low 사이의 갭
- 원리: 세력은 FVG로 돌아와 포지션을 채움
- **Bullish FVG**: 상승 임밸런스 → 지지로 작용
- **Bearish FVG**: 하락 임밸런스 → 저항으로 작용
- **자동화 포인트**: 3캔들 패턴으로 FVG 감지 가능, 가격이 FVG 진입 시 신호 생성

#### 4. PO3 (Power of Three) = AMD 패턴
- **Accumulation (축적)**: 세력이 물량 모으는 구간 (횡보, 낮은 변동성)
- **Manipulation (조작)**: 가짜 방향 돌파로 개미 스탑 터트림 (리테일 트랩)
- **Distribution (분배)**: 실제 방향으로 강하게 이동
- 중요성: 가짜 돌파(Fake-out)에 당하지 않는 핵심 패턴
- **자동화 포인트**: 횡보 후 급격한 반전 → 진짜 방향 추적 가능

#### 5. 균형가격 (BPR, Balanced Price Range)
- 정의: Bullish FVG와 Bearish FVG가 겹치는 구간
- 특징: 저항은 강하고 지지는 단단한 A+ 진입 타점
- **자동화 포인트**: 두 FVG의 겹침 감지로 고확률 진입 존 마킹

#### 6. IFVG (Inverse Fair Value Gap, 반전 FVG)
- FVG가 충전(미팅)된 후 반전 신호로 작동하는 구간
- BPR과 함께 사용 시 최고 타점 도출

#### 7. 고저항 vs 저저항 유동성
- **고저항 유동성**: 강한 레벨에서의 유동성 → 큰 반전 기대
- **저저항 유동성**: 약한 레벨의 유동성 → 그냥 통과 가능성 높음
- 진입 기준: 저저항 유동성 공간에서 고저항 유동성 타겟

### 7가지 실전 체크리스트
전업 트레이더의 차트 분석 순서 (이 순서대로만 하면 실수 90% 감소):

```
1. 유동성(Liquidity) 위치 확인
2. 유동성 상태(축적 vs 스윕) 파악
3. 파워 오브 3 (PO3) 단계 파악
4. FVG 형성 확인
5. FVG 반응(리스펙트 vs 리젝션) 관찰
6. 균형 가격(BPR) 형성 여부 체크
7. 최종 진입 결정 (A+ 셋업)
```

### 킬존 (Kill Zone)
- 정의: 기관이 실제로 거래하는 특정 시간대
- 중요성: 킬존 외에는 거래 자제
- 타입:
  - **아시아 킬존**: 런던 오픈 전 방향성 세팅
  - **런던 킬존**: 유럽 시장 오픈 (고변동성)
  - **뉴욕 킬존**: 미국 오픈 (최고 유동성)
- **자동화 포인트**: 시간 필터로 특정 시간대에만 신호 생성

### 핵심 개념

#### 1. 킬존 세부 (3가지)
- **아시아 킬존**: 한국 시간 기준 야간
- **런던 킬존**: 런던 오픈 (유럽 세력 진입)
- **뉴욕 킬존**: 뉴욕 오픈 (기관 최대 활성)
- **뉴욕 킬존 3분할**: 오픈 직후 30분이 가장 중요

#### 2. Opening Range (OR) & Initial Range (IR)
- **OR (Opening Range)**: 뉴욕 오픈 직후 30분간의 고/저점
- **IR (Initial Range)**: 첫 캔들의 고/저 범위
- 원리: OR 고/저점이 당일 방향의 기준선 역할
- OR 위로 이탈 → 상승 세션, OR 아래로 이탈 → 하락 세션
- **자동화 포인트**: 오픈 후 30분 OR 박스 자동 계산, 이탈 방향으로 신호

#### 3. 4가지 진입 체크리스트 (YES가 4번 나와야 진입)
```
질문 1: 현재 킬존 안에 있는가?
질문 2: 유동성이 스윕(소탕)되었는가?
질문 3: FVG가 형성되었는가?
질문 4: 가격이 FVG에 반응하는가?
→ 4개 모두 YES → A+ 셋업
```

#### 4. HP FVG (High Probability FVG, 고확률 갭)
- 정의: 유동성 흡수에 기여한 FVG
- 일반 FVG와 차이: 유동성 스윕 직후 형성된 FVG가 훨씬 신뢰도 높음
- **자동화 포인트**: 유동성 스윕 이후 형성된 FVG에만 가중치 부여

#### 5. 꼬리 vs 몸통
- FVG의 경계를 꼬리 기준으로 볼지 몸통 기준으로 볼지에 따라 진입 정확도 달라짐
- 꼬리 FVG: 더 보수적인 진입
- 몸통 FVG: 더 공격적인 진입

#### 6. 세션 간 유동성 흐름
- 아시아 세션이 만든 유동성을 런던이 스윕
- 런던이 만든 유동성을 뉴욕이 스윕하는 패턴
- **자동화 포인트**: 전 세션 고/저점을 다음 세션의 유동성 타겟으로 설정

#### 7. 프리미엄 유동성
- 가격 범위의 상위 50% 이상 = 프리미엄 존 (매도 유리)
- 가격 범위의 하위 50% 이하 = 디스카운트 존 (매수 유리)
- **자동화 포인트**: 스윙 고/저점의 중간값(50%)을 기준으로 프리미엄/디스카운트 구분

### 추론되는 핵심 내용 (중급 차트 #19 손절 전략)

#### 손절에 대한 프로 트레이더의 관점
- 손절은 "비용"이 아니라 "확률 게임의 필수 요소"
- 손절 없이는 리스크 관리 불가 → 장기 생존 불가
- "손절을 씹어 먹는다" = 손절을 두려워하지 않고 시스템화

#### 일반적인 손절 배치 원칙 (ICT/차트 기반)
- **구조 기반 손절**: 직전 스윙 고점/저점 너머에 손절 배치
- **FVG 기반 손절**: FVG 무효화 지점(반대편 경계)에 손절
- **Order Block 기반 손절**: 오더블록이 깨지는 지점에 손절
- **ATR 기반 손절**: 평균 변동폭 × 배수로 손절폭 설정

#### 손절 배치 실수 유형
1. 너무 좁은 손절 → 노이즈에 의한 불필요한 손절
2. 너무 넓은 손절 → 리스크/리워드 비율 악화
3. 라운드 넘버에 손절 → 기관이 알고 있는 곳, 스윕 당함
4. 이동 평균선에 손절 → 모두가 아는 곳 = 유동성 집중 = 위험

#### 스마트한 손절 배치
- 유동성 클러스터(모두가 놓는 곳) 바로 너머 배치하지 말 것
- 구조적 무효화 지점(Structure Invalidation Point)에 배치
- 포지션 크기와 손절 거리의 반비례 관계 활용

### ICT 핵심 요약 (700개 강의 압축)

#### 챕터 1 (01:09~05:49): 시장 구조 이해
- **Market Structure**: HH(Higher High), HL(Higher Low) = 상승 구조
- **BOS (Break of Structure)**: 구조 돌파 → 추세 확인
- **ChoCH (Change of Character)**: 성격 변화 → 추세 전환 신호
- 스윙 포인트 파악이 모든 분석의 기초
- **자동화 포인트**: HH/HL/LH/LL 패턴 자동 감지 → 추세 방향 확정

#### 챕터 2 (05:49~10:04): 유동성과 오더블록
- **Order Block (OB)**: 기관이 대량 주문을 넣은 마지막 반대색 캔들
  - Bullish OB: 상승 직전의 마지막 하락 캔들 (낮은 위치)
  - Bearish OB: 하락 직전의 마지막 상승 캔들 (높은 위치)
- **Breaker Block**: 무효화된 OB가 반대 방향 지지/저항으로 전환
- **Mitigation Block**: OB의 일부만 충전된 후 반전

#### 챕터 3 (10:04~13:27): FVG & 임밸런스
- **FVG (Fair Value Gap)**: 3캔들 임밸런스 구간
- **Void**: 극단적 임밸런스 (캔들 전체가 갭)
- **SIBI/BISI**:
  - SIBI (Sell-Side Imbalance, Buy-Side Inefficiency)
  - BISI (Buy-Side Imbalance, Sell-Side Inefficiency)
- **자동화 포인트**: FVG 방향성(Bullish/Bearish) + 크기 + 충전 비율 추적

#### 챕터 4 (13:27~18:00): 타임프레임 상관관계
- HTF (Higher Time Frame) → LTF (Lower Time Frame) 하향식 분석
- HTF에서 방향 결정 → LTF에서 정밀 진입
- **Confluence (컨플루언스)**: 여러 레벨이 겹치는 곳이 최고 타점
  - HTF OB + LTF FVG + 킬존 시간 = A+ 셋업
- **자동화 포인트**: 멀티 타임프레임 신호 동시 확인 로직

- **리스크 관리 우선**: 수익보다 손실 제한이 먼저

### 2. 전략 엔진 구현 로직 제안

#### FVG 감지 알고리즘
```python
def detect_fvg(candles):
    """
    3캔들 패턴으로 FVG 감지
    - Bullish FVG: candle[0].high < candle[2].low
    - Bearish FVG: candle[0].low > candle[2].high
    """
    fvgs = []
    for i in range(len(candles) - 2):
        c1, c2, c3 = candles[i], candles[i+1], candles[i+2]
        if c1.high < c3.low:  # Bullish FVG
            fvgs.append({'type': 'bullish', 'top': c3.low, 'bottom': c1.high, 'index': i+1})
        elif c1.low > c3.high:  # Bearish FVG
            fvgs.append({'type': 'bearish', 'top': c1.low, 'bottom': c3.high, 'index': i+1})
    return fvgs
```

#### 유동성 존 마킹
```python
def find_liquidity_zones(candles, lookback=20):
    """
    스윙 고/저점 기반 유동성 존 감지
    - Equal Highs (EQH): 같은 레벨의 고점 집중 → Buy-Side Liquidity
    - Equal Lows (EQL): 같은 레벨의 저점 집중 → Sell-Side Liquidity
    """
    swing_highs = [c.high for c in candles[-lookback:]]
    swing_lows = [c.low for c in candles[-lookback:]]
    return {
        'buy_side': max(swing_highs),   # 최근 고점 = 매수 유동성 타겟
        'sell_side': min(swing_lows),   # 최근 저점 = 매도 유동성 타겟
    }
```

### 3. 손절 전략 원칙 (비디오 3 기반)

#### 구조 기반 손절 (권장)
- 진입 방향의 반대 구조 무효화 지점에 배치
- Long 진입 시: 직전 HL(Higher Low) 아래에 손절
- Short 진입 시: 직전 LH(Lower High) 위에 손절

#### FVG 기반 손절
- Bullish FVG 진입 시: FVG 하단 아래에 손절
- 손절 거리 = FVG 크기

#### 절대 피해야 할 손절 위치
- 라운드 넘버 (1000, 5000, 10000 등)
- 직전 봉 고/저점 바로 위/아래 (모두가 놓는 위치 = 유동성 스윕 대상)

### 4. 멀티 타임프레임 분석 체계

```
HTF (일봉/주봉): 전체 방향 결정 (Bullish/Bearish bias)
    ↓
MTF (1시간/4시간): Order Block, FVG 식별
    ↓
LTF (15분/5분): 정밀 진입 타점, 킬존 내 FVG 반응 확인
```

### 5. "차트 없는 트레이딩"의 자동화 의미 (비디오 5 기반)

- 사람이 느끼는 "차트 없음" = 미리 정의된 조건만 체크
- 자동매매 시스템 = 이 조건들을 기계적으로 체크하는 시스템
- 핵심: **과도한 지표 지양, 명확한 규칙 2~3개에 집중**
- 단순한 시스템이 복잡한 시스템보다 실전에서 강함
