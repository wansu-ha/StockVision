# 수집 가능 데이터 인벤토리

> 작성일: 2026-03-07

각 데이터 소스별로 수집할 수 있는 모든 데이터를 정리한다.
DataProvider spec 확장 및 AI 분석 파이프라인 설계의 기초 자료.

## 우선순위 제안

### P0 — v1 (현재 DataProvider spec 범위)
- 일봉 OHLCV (yfinance → KIS/키움)
- 현재가 (KIS/키움 REST)
- 지수/환율 (yfinance, 이미 수집 중)

### P1 — v1 확장
- 호가 10호가 (KIS/키움, 매매 판단 필수)
- 분봉 OHLCV (KIS/키움)
- 투자자별 매매동향 (키움 `frgnistt` / KRX)
- 재무제표 (DART OpenAPI)
- 배당 정보 (yfinance + DART)
- 업종 지수 (키움 `sect` + `0J` WS)

### P2 — v2 (AI 분석 파이프라인)
- 기업 공시 (DART)
- 거시경제 (한국은행 ECOS)
- 공매도/대차 (키움 `shsa`/`slb`)
- 테마/순위 (키움 `thme`/`rkinfo`)
- VI 발동 이벤트 (키움 `1h` WS)

### P3 — v2+ (ContentProvider 영역)
- 뉴스 크롤링
- 커뮤니티 감성 (네이버 종토방, DC)
- 검색량 추이 (Google Trends)
- 증권사 리포트/목표가
