# 전략 빌더 UI/UX 벤치마킹

경쟁 제품의 전략 생성 방식과 UI/UX 핵심 차별점 분석.

---

## 1. Composer.trade — AI 자연어 전략 빌더

**전략 생성 방식**: 자연어 입력 → AI가 전략 자동 생성 + No-code 비주얼 에디터 병행

**UI/UX 특징**
- "Trade with AI" (2025.10): 목표를 평문으로 설명하면 60초 내 백테스트 완료된 실행 가능 전략 생성
- 자체 DSL(proprietary trading language) 위에 LLM을 얹어 서브초(sub-second) 백테스트 구현
- 커뮤니티 공개 "Playbook" 브라우징 → 원클릭 복사·실행
- 주식, ETF, 크립토 통합 지원

**타깃**: 코딩 불가 리테일 투자자 (기관급 알고리즘을 민주화)

**StockVision에서 배울 점**
- AI 입력 → 즉시 백테스트 결과 제시 플로우가 핵심 UX. "전략 설명 → 결과 확인" 단계를 최대한 짧게.
- 전략 공유(공개 플레이북) 기능은 플랫폼 네트워크 효과를 만듦.

---

## 2. TradingView Pine Script — 코드 기반 전략 에디터

**전략 생성 방식**: 전용 스크립팅 언어(Pine Script) 코드 작성

**UI/UX 특징**
- 2025년 에디터 패널 수직 이동 (우측 사이드패널) → 차트와 코드 동시 확인
- Pine Profiler: 코드 라인별 실행 시간 표시 → 병목 즉시 파악
- Strategy Tester 탭: 매매 마커를 차트에 직접 오버레이
- 커뮤니티 스크립트 공개 마켓플레이스 (수만 개 공개 전략)
- v6로 업그레이드, 로컬 스코프 제한(550개) 완전 제거

**타깃**: 개발자·퀀트 트레이더 (기술적 깊이 우선)

**StockVision에서 배울 점**
- 백테스트 결과를 차트 위에 직접 시각화(진입/청산 화살표)하는 방식이 직관적.
- 에디터와 차트를 같은 화면에 배치하면 전략 디버깅 속도가 빨라짐.

---

## 3. 3Commas — DCA 봇 폼 빌더

**전략 생성 방식**: 단계별 폼 입력 (파라미터 기반) + 프리셋 선택

**UI/UX 특징**
- Classic / Advanced 2가지 레이아웃 선택 (차트 필요 여부에 따라)
- 3열 레이아웃: 폼 + 차트 + 주문 테이블 동시 표시
- Strategy Presets: 시나리오(Classic Trading, Super Power 등) 선택 → 설정 자동 입력
- 차트 위에서 TP/SL/분할매수 주문선을 드래그로 직접 조정
- Market Data Insight: 기술적 필터로 최적 진입 타이밍 탐색
- 100+ 페어 동시 모니터링, 봇 복제 기능

**타깃**: 크립토 DCA·그리드 전략 사용자 (중급)

**StockVision에서 배울 점**
- 프리셋 → 파라미터 자동 채움 패턴은 초보자 진입 장벽을 크게 낮춤.
- TP/SL을 차트에서 직접 드래그하는 인터랙션은 숫자 입력보다 직관적.

---

## 4. TrendSpider — 비주얼 조건 빌더 + AI 조건 생성

**전략 생성 방식**: 드롭다운 비주얼 조건 편집기 (No-code) + AI 자연어 보조

**UI/UX 특징**
- 조건을 드롭다운 메뉴로 선택 ("50일 MA가 200일 MA를 상향 돌파")
- 2025.09부터 자연어(음성 포함) 조건 입력이 기본 UI로 전환
- 전략 결과를 차트에 진입/청산 화살표로 시각화 + 드로우다운 곡선 오버레이
- Strategy Variance Test: 복수 전략·심볼·타임프레임 일괄 백테스트 → 스프레드시트 결과
- No-go 플래그: 수학적으로 수익 불가능한 전략 즉시 경고
- Price Behavior Explorer: 개선 포인트 KPI 제시
- 50년치 데이터, 주식·선물·크립토·FX 지원

**타깃**: 기술적 분석 기반 트레이더 (중급~고급)

**StockVision에서 배울 점**
- "No-go 플래그"처럼 전략 실행 전 수익 가능성을 사전 경고하는 가이드레일이 UX 신뢰도를 높임.
- 자연어 → 비주얼 조건으로 변환하는 하이브리드 방식이 접근성과 정밀도를 동시에 만족.

---

## 5. 키움증권 영웅문 — 조건검색 + 수식관리자

**전략 생성 방식**: 수식(DSL) 직접 작성 + 지표 파라미터 폼 입력 (HTS 기반)

**UI/UX 특징**
- 수식관리자: 기술적 지표 함수식 직접 편집, 오류검증 내장
- 조건검색식: 이동평균 크로스, MACD 크로스 등 패턴을 함수식으로 정의
- 전략 적용 후 "시험 적용"(수익성 측정) vs "주문창 연동"(실매매 연결) 2모드
- 조건검색식 서버 저장 (PC 독립)
- 종목연동: 열린 화면들 간 선택 종목 자동 연동
- 모바일(영웅문S#)에서도 실시간 조건검색 지원

**타깃**: 국내 주식 개인 투자자·시스템 트레이더 (중급~전문가)

**StockVision에서 배울 점**
- 조건 설정 후 "시험 적용"과 "실매매 연동"을 명확히 분리하는 모드 구분은 안전성 측면에서 중요.
- 국내 사용자는 한국어 지표명과 조건식 표기에 이미 익숙 → UI 용어를 영웅문 관행에 맞추면 학습 비용 절감.

## 핵심 인사이트 요약

| 제품 | 입력 방식 | 핵심 강점 |
|------|-----------|-----------|
| Composer | 자연어 → 전략 | 60초 백테스트, 플레이북 공유 |
| TradingView | 코드(Pine Script) | 차트-에디터 통합, 커뮤니티 마켓 |
| 3Commas | 폼 + 프리셋 | 차트 드래그 TP/SL, 프리셋 자동입력 |
| TrendSpider | 비주얼 + 자연어 | No-go 경고, 변형 테스트 |
| 영웅문 | 수식 DSL | 한국 관행 준수, 시험/실매매 모드 분리 |

### StockVision 적용 우선순위

1. **자연어 → 전략 변환** (Composer 방식): AI가 전략 초안을 생성하고 사용자가 파라미터만 조정.
2. **차트 위 시각화** (TradingView + TrendSpider): 백테스트 진입/청산 마커와 드로우다운 곡선 오버레이.
3. **프리셋 → 자동입력** (3Commas): 초보자용 전략 템플릿 선택 → 파라미터 자동 채움.
4. **시험/실매매 모드 분리** (영웅문): "백테스트 모드"와 "실주문 연동 모드"를 UI에서 명확히 구분.
5. **사전 가드레일** (TrendSpider): 전략 저장 전 리스크/수익 가능성 경고 표시.
