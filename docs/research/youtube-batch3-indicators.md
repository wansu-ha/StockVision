### 하이킨아시 캔들 패턴 3가지

| 패턴 | 조건 | 신호 |
|------|------|------|
| 강세 양봉 | 녹색 몸통 + 아래꼬리 없거나 짧음 | 상승 추세 지속 |
| 강세 음봉 | 빨간 몸통 + 윗꼬리 없거나 짧음 | 하락 추세 지속 |
| 도지 | 몸통 작음 + 위아래 꼬리 길음 | 추세 전환 임박 |

### 구현 가능한 규칙

**매수 진입 조건 (추세 전환 후 롱)**:
```
1. 하락 추세 중 도지 캔들 2개 이상 연속 출현
2. 양봉 전환 확인 (윗꼬리 짧고 몸통 큰 강세 양봉)
3. 손절: 양봉 저점(꼬리 하단)
4. 익절1: 직전 저항선 / 조정 구간 상단
5. 익절2(스윙): 양봉 몸통 작아지거나 꼬리 길어지거나 음봉 전환 시
```

**매도 진입 조건 (추세 전환 후 숏)**:
```
1. 상승 추세 중 도지 캔들 2개 이상 출현
2. 음봉 전환 확인 (윗꼬리 없고 몸통 커지는 음봉)
3. 손절: 음봉 고점
4. 익절: 직전 지지선
```

**추세 강도 필터**:
```
- 꼬리 없는 양봉 연속 → 강한 상승 추세 (매수 유지)
- 꼬리 없는 음봉 연속 → 강한 하락 추세 (매도 유지)
- 몸통 점점 작아지고 꼬리 길어짐 → 추세 약화, 익절 준비
- 비슷한 크기 캔들 횡보 → 박스권 진입, 대기
```

### 중요 주의사항
- 하이킨아시는 **예측용이 아니라 추세 확인용**
- 정확한 진입가/손절가를 잡는데 단독 사용 부적합 (시가/종가가 실시간과 불일치)
- **권장 타임프레임**: 일봉 이상 큰 프레임 (노이즈 제거 효과 극대화)
- **추천 조합**: 스토캐스틱 + 거래량과 함께 사용

### StockVision 구현 포인트
- 하이킨아시 캔들 계산: `HA_Close = (O+H+L+C)/4`, `HA_Open = (이전HA_Open + 이전HA_Close)/2`
- 꼬리 길이 계산: 상단꼬리 = `H - max(HA_Open, HA_Close)`, 하단꼬리 = `min(HA_Open, HA_Close) - L`
- 패턴 감지: 꼬리/몸통 비율 기준점 필요 (예: 꼬리 < 몸통 * 0.1 = "꼬리 없음")

### RSI 고급 활용법

#### 1. 기본 매매 전략 (업그레이드 버전)
```
기본 매수: RSI 30 이하 진입 X
업그레이드: RSI가 30선을 상향 돌파할 때 매수 (침체권 탈출 확인)
업그레이드: RSI가 70 이상 진입해도 70선 이탈 전까지 홀딩 (추세 따라가기)
50선 매도: RSI가 30 상향 돌파 후 50선 도달 시 단기 매도 대응
```

#### 2. RSI 50선 지지/저항 전략
```
매수 신호: RSI 50선 상향 돌파 (중기 상승 추세 전환)
매도 신호: RSI 50선 하향 이탈 (하락 추세 강화)
이탈 후 반등 → 50선에서 저항 받으면 매도 유효
```

**중요**: RSI 50선이 저항 작용을 계속하는 종목 = 하락 추세 강도 강함 → 매매 제외

#### 3. RSI 페일류 스윙(Failure Swing) 전략

**상승 페일류 스윙 (매수)**:
```
조건: RSI 침체권(30 이하)에서 여러 번 상향 돌파 시도 후 다시 하락 (페일류) 반복
확인: MACD선이 시그널선 아래에 있으면 페일류
진짜 돌파: MACD선이 시그널선 위에 있을 때 RSI 30 상향 돌파 → 매수
```

**하락 페일류 스윙 (매도/홀딩 청산)**:
```
조건: RSI 과열권(70 이상)에서 여러 번 하향 이탈 시도 후 다시 상승 (페일류) 반복
확인: MACD선이 시그널선 위에 있으면 페일류 (아직 상승 중)
진짜 이탈: MACD 데드크로스 발생 시 → 강력 매도 신호
```

#### 4. RSI + MACD 복합 매수 전략 (핵심)

**1차 매수 타점**:
```
조건 1: RSI 50선 상향 돌파
조건 2: MACD 영선(0선) 상향 돌파 + 골든크로스 동시 발생
→ 중기 상승 추세 전환 신호, 매수 진입
```

**2차 매수 타점 (더 강력)**:
```
조건 1: RSI가 70 이상 과열권 진입 후 페일류 없이 계속 상승 (MACD선 > 시그널선 확인)
조건 2: RSI 파동 관점에서 저점이 높아지며 돌파
조건 3: MACD가 낮은 위치에서 상승 시작 (상승 공간 충분)
→ MACD 직전 고점 돌파 시 매수
```

### 키움증권 조건 검색식 (MACD 기법)
```
조건: MACD 영선(0선) 상향 돌파 (최근 1~5봉 이내)
추가조건: 52주 고점 대비 -10%~0% (신고가 근접)
추가조건: 거래대금 상위 100위 이내
→ 검색 결과가 한국 주식 유망 종목 후보
```

### 중요 맥락: RSI 페일류 발생 원인
- RSI 기본값 14일 기준, 15일째 가장 오래된 가격 제외되면서 계산값 변동
- 이전 하락폭이 제외되면 현재 가격이 그대로여도 RSI 상승 → 페일류 발생
- **이 현상을 이해하고 MACD로 진짜 돌파인지 구별**

### StockVision 구현 포인트
```python
# MACD 영선 돌파 조건 (조건 검색용)
def macd_zero_cross_up(macd_values, n_bars=5):
    """최근 n_bars 내 MACD 영선 상향 돌파 확인"""
    for i in range(n_bars):
        if macd_values[-i-2] < 0 and macd_values[-i-1] >= 0:
            return True
    return False

# RSI 50선 돌파 조건
def rsi_50_cross_up(rsi_values):
    return rsi_values[-2] < 50 and rsi_values[-1] >= 50
```

### StockVision 구현 포인트
```python
# VWAP/VWMA 계산
def vwma(prices, volumes, period):
    """거래량 가중 이동평균"""
    pv = [p * v for p, v in zip(prices[-period:], volumes[-period:])]
    return sum(pv) / sum(volumes[-period:])

# 이평선 정배열 확인
def is_golden_alignment(ma_short, ma_mid, ma_long):
    """정배열: 단기 > 중기 > 장기"""
    return ma_short > ma_mid > ma_long

# 이평선 역배열 → 정배열 전환
def is_alignment_reversing(prev_short, prev_mid, curr_short, curr_mid):
    was_bearish = prev_short < prev_mid
    is_bullish = curr_short > curr_mid
    return was_bearish and is_bullish
```

### WB 매매 규칙

#### 변곡 매매 (반전)

**매도 변곡**:
```
조건 1: 캔들이 R1 상단 + 기본 BB 상단 동시 터치
조건 2: 캔들이 두 밴드를 돌파 못하고 꼬리 달린 음봉 마감
→ 음봉 마감 확인 즉시 매도 진입

주의: 밴드 돌파한 것처럼 보여도 앞의 매물대(전고점)를 못 뚫으면 가짜 돌파
→ 다음 캔들에서 음봉 마감으로 재확인 후 진입 (이종 변곡)
```

**매수 변곡**:
```
조건 1: 캔들이 R1 하단 + 기본 BB 하단 동시 터치
조건 2: 윗꼬리 달린 양봉 마감 or 아래꼬리 달린 반등 캔들 마감
→ 양봉 마감 확인 즉시 매수 진입
```

#### 돌파 매매 (브레이크아웃)

**매수 돌파**:
```
조건 1: 두 밴드(R1 + 기본 BB) 상단 모두 돌파
조건 2: 앞에 저항 매물대(전고점) 도 돌파
조건 3: 종가가 두 밴드 위에서 마감
→ 돌파 캔들 종가에서 매수

이후 추세 매매:
→ 첫 돌파 후 눌림(되돌림) 발생 시 매수 진입
→ 추세 초반에만 진입 (추세 끝나고 진입 금지)
```

**손절/익절 기준**:
```
변곡 매매:
  손절: 진입 기준 밴드 하단 이탈 시
  익절: 반대편 밴드 (하단 진입 → 상단이 목표)
  손익비: 기본 1:3 구조

돌파 매매:
  손절: 밴드 안으로 재진입 시
  익절: 추세가 약화될 때 (이전 고점, 저항선)
```

### R1(기간 4, 표준편차 4)을 사용하는 이유
```
기간 4 = 일봉 기준 1주(5거래일) 보다 약간 짧은 기간
       = 1시간봉 기준 4시간봉(많은 트레이더가 참고하는 프레임)
표준편차 4 = 극단적 이탈 감지 (더 확실한 신호만 포착)
```

### StockVision 구현 포인트
```python
# WB 볼린저밴드 계산
def double_bollinger(prices, opens):
    # 기본 BB (흰색)
    bb_mid = sma(prices, 20)
    bb_std = std(prices, 20)
    bb_upper = bb_mid + 2 * bb_std
    bb_lower = bb_mid - 2 * bb_std

    # R1 (빨간색) - Open 기반
    r1_mid = sma(opens, 4)
    r1_std = std(opens, 4)
    r1_upper = r1_mid + 4 * r1_std
    r1_lower = r1_mid - 4 * r1_std

    return bb_upper, bb_lower, r1_upper, r1_lower

# 변곡 매도 신호
def wb_reversal_sell(price, high, close, r1_upper, bb_upper, prev_high):
    # 두 밴드 동시 터치
    touched_both = high >= r1_upper and high >= bb_upper
    # 음봉 마감 (꼬리 후 내려옴)
    bearish_close = close < r1_upper and close < bb_upper
    return touched_both and bearish_close

# 진짜 돌파 확인
def is_real_breakout(close, r1_upper, bb_upper, prev_resistance):
    above_both = close > r1_upper and close > bb_upper
    broke_resistance = close > prev_resistance
    return above_both and broke_resistance
```
