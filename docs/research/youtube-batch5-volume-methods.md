## 영상 1 — 700억을 만든 거래량 '진짜' 매매법! (스티븐 덕스 매매법)

**출처:** 캡틴의 인생게임 | https://www.youtube.com/watch?v=zUY-ChZHvUE

### 핵심 내용 요약

미국 트레이더 스티븐 덕스(Steven Dux)의 거래량 기반 매매 시스템을 소개. 20대 초반에 4,800만 달러(~700억 원)를 벌어들인 데이트레이더. 소형주(시가총액 낮고 변동성 높은 종목) 단기 급등락 패턴 공략.

**핵심 철학:**
- "캔들 모양은 세력이 속임수로 만들 수 있지만, 수천억의 거래량은 절대 숨길 수 없다"
- 감정 배제, 확률과 통계 기반 기계적 매매
- 보조 지표 대신 거래량 하나로 세력 진입·이탈 추적

### 코드화 가능한 거래량 매매 규칙

#### 지표 세팅
- 거래량 MA 기간: **10** (기본값 20에서 변경)
- 거래량 막대 투명도: 80%
- MA선 색상: 흰색, 굵기 2~3단계

#### 진입 조건 (매수)
1. **거래량 폭발 확인**: 거래량이 MA선을 급격히 돌파 (세력 진입 신호)
2. **박스권 형성 대기**: 급등 이후 횡보 구간 형성될 때까지 기다림
3. **거래량 소진 확인**: 박스권 횡보 중 거래량이 MA선 아래로 떨어짐 (연료 소진)
4. **패럴렐 채널 활용**: 박스권에 패럴렐 채널 설정 → 중심선 위에서 가격이 유지되는 경우
5. **돌파 캔들 확인**: 저항선을 완전히 이탈하는 **양봉 캔들 몸통**이 나올 때 매수

#### 진입 조건 (매도/공매도)
1. 거래량 소진 상태에서 **지지선을 완전히 이탈하는 음봉 몸통** 출현
2. 박스권 중심선을 가격이 돌파하지 못하고 하단을 맴도는 상태
3. 꼬리만 지지선 이탈 → 기다림 (몸통 이탈 확인 필요)

#### 손익비 원칙
- **손익비 1:3** 고수 (손절 짧게, 익절 길게)
- 손절: 박스권 상단 저항선 또는 직전 전고점
- 익절: 손절폭의 3배 거리

#### 적용 가능 시장
- 주식, 코인, 해외 선물 모든 시장 적용 가능
- 선호 타임프레임: 1분봉 (데이트레이딩)

## 영상 2 — 거래량 확실하게 이해시켜 드립니다 (차트프로 #38)

**출처:** 차트프로 | https://www.youtube.com/watch?v=cr1Imugb2gg

### 핵심 내용 요약

거래량과 주가 움직임의 5가지 패턴 분류. "차트는 외우는 게 아니라 이해하는 영역"

### 거래량의 3가지 역할
1. **신뢰도 인증**: 거래량이 많은 가격 = 많은 사람이 합의한 가격 → 신뢰 가능
2. **방향성 신호**: 거래량 증가 → 위든 아래든 움직임 발생 (방향 자체는 확인 필요)
3. **결과 가속화**: 거래량 증가 = 빠른 결과 도출 → 멘탈 관리 용이, 뇌동 매매 방지

### 코드화 가능한 5가지 거래량-주가 패턴

| 패턴 | 거래량 | 주가 | 해석 | 대응 |
|------|--------|------|------|------|
| 패턴 A | 증가 | 상승 | 매수 의지 강함, 매물대 많음, 매집 안 됨 | 상승 확률 높음 (고점 주의) |
| 패턴 B | 증가 | 하락 | 매도 의지 강함, 속임수 패턴 | 즉시 매도 탈출 |
| 패턴 C | 감소 | 상승 | 세력 관리 차트, 매물대 적음, 매집 완료 | **적극 매수** (가장 이상적) |
| 패턴 D | 감소 | 하락 | 세력이 천천히 물량 넘김, 희망고문 | 하락 확률 높음, 조심 |
| 패턴 E | 감소 | 횡보 | 거래량 폭발 후 기준가 위에서 조정 | **2차 상승 최적 매수 타이밍** |

#### 패턴 C 특징 (세력 관리 차트 식별 조건)
- 장대 양봉 이전 캔들 크기에 비해 이후 캔들이 현저히 작음
- 오전 상승 → 오후 하락 마감 음봉 반복
- 연속 같은 색깔 캔들 10일 이상 → 의도적 관리 신호

#### 매수 기준가 (허리 라인) 개념
- 거래량 폭발 캔들의 종가 = 기준가 (허리)
- 기준가 위에서 조정받는 것이 이상적
- 기준가 이탈 + 전저점 이탈 → 손절 라인

#### 거래량 없는 구간 매매 금지
- 거래량 없는 횡보 구간: 손실 확률 높음 (지속적 흔들림 발생)
- 거래량이 터진 후 확인된 방향성에서만 매매

### 호가창 해석 (의지 차이 개념)
- 주가 상승 ≠ 매수자 수 > 매도자 수 (매수/매도 수는 항상 동일)
- 주가 상승 = **매수자의 의지(긴박성)가 매도자보다 강함**
- 급등 시: 매수자가 시장가로 즉시 체결, 매도 잔량은 상대적으로 유지

## 영상 3 — 소액으로 반드시 벌기 위한 매매법 (키움영웅전 수상자 인터뷰)

**출처:** 키움영웅전 | https://www.youtube.com/watch?v=WpeW12oPBRk

**인터뷰이:** 방탄 (2025 키움영웅전 리그 1위, 회전율 131,181%, 수익률 301%)
- 운용 시드: 250만 원(대회) → 현재 500~600만 원
- 전업 트레이더, 주식 경력 3년

### 소액 매매 핵심 원칙

#### 1. 재료 기반 스캘핑
- "모든 주식의 힘은 재료에서 나온다"
- 텔레그램, HTS 시황창(속보/단독) 실시간 모니터링
- 뉴스 제목만 보고 순식간에 판단: "이게 사람들이 달려들 재료인가?"
- 기본적 분석(실적 등) 무시 → 차트 + 호가 기술적 분석 집중

#### 2. 종목 선정 프로세스
1. 실시간 조회순위 1~20위 모니터링 (단, 무조건 따라가지 않음)
2. 텔레그램 뉴스 → 재료 확인
3. HTS 시황창 속보/특징주 모니터링
4. 네이버 뉴스 검색 (텔레그램에 없는 경우 보완)
5. 유튜브 장전/장후 시황 채널 활용 (심플관심종목TV)

#### 3. 진입 타이밍
- 돌파 vs 눌림보다 **재료 확신**이 더 중요
- 20~25% 급등 중이라도 재료 확신 시 진입
- 분할 진입: 1/3씩 3회 분할 (첫 매수 후 상승 확인 시 불타기)
- **물타기 절대 금지**, **불타기만 허용**

#### 4. 매도 타이밍
- 큰 물량이 들어왔다가 즉시 더 오르지 않으면 → 매도 준비
- 전고점 근처에서 주로 매도
- 전고점 돌파 실패 시 → 즉시 매도 (실망 매물 선점)
- "큰 돈이 들어와서 같이 탔지만, 또 큰 돈이 들어올 때는 내가 나갈 타이밍"

#### 5. 손절 원칙
- **매수 직후 오르지 않으면 손절 준비** (커서를 1호가 아래 매도에 대기)
- 파란불 켜지는 순간 = 뭔가 잘못된 신호
- 잔 손절 다수 + 큰 수익 소수 = 순 수익 양수
- 목표 수익 미달 시 오버트레이딩 금지

#### 6. 수급 확인 도구
- 분봉 거래대금 수식 관리자 설정 (각 봉의 거래대금 표시)
- 순간체결량: **3,000만 원 이상만 표시** (잡음 제거)
- 시총 1,000~2,000억 종목 기준: 분봉 당 20~30억 이상 = 의미 있는 수급
- 대형주(한화 시리즈 등): 분봉 당 70~80억 이상 기준

#### 7. 차트 설정 (미니멀 원칙)
- 기본 이동평균선 (5, 10, 20, 60, 120일/분)
- 분봉 거래대금 (수식 관리자)
- 전고점 수평선
- **보조지표 일체 없음** ("남들과 같은 악보를 봐야 같은 연주 가능")

#### 8. 시드 관리
- 소액(500만 원)으로 충분 → 회전율로 커버
- 시드 단계적 증가 (500 → 700 → 1,000…, 각 단계 적응 필수)
- 신용·미수 사용: **재료 확신 시에만** 풀 활용
- 생활비 출금 후 수익금으로만 시드 확대

#### 9. 시간대별 전략
- 오전장: 소액 + 다수 매매 (손절 폭 짧게)
- 장 마감 전 종가 배팅: 어떤 섹터에 돈 쏠리는지 파악 후 대장주 매수
- 시간외 거래: 비중 크게 실어 메인 수익 추구
- 장이 안 좋을 때: 짧은 손절로 오히려 회전율 높이는 전략 유효

#### 10. 군중 심리 활용
- 1분봉 사용 이유: "대부분의 단기 트레이더가 1분봉 보기 때문"
- 같은 보조지표 = 같은 반응 → 기본 이평선만 사용
- "서로 다른 악보를 보면 같은 연주 불가"

## 영상 4 — 실제로 주가조작은 어떤식으로 이루어질까

**출처:** 은근한 잡다한 지식 | https://www.youtube.com/watch?v=SfxCgc-5cm0

### 주가조작 메커니즘 정리

#### 작전주 타겟 특성 (위험 종목 식별 기준)
- **시가총액 소형** (조작에 필요한 자금 최소화)
- **거래량 적음** (매수·매도 타이밍 통제 가능)
- 저가주 (낮은 주가 = 적은 자금으로 지분 확보)

#### 주가조작 단계 (4단계)

**1단계 — 매집 (시장조성)**
- 브로커가 타겟 종목 선정
- 쩐주(큰손)로부터 자금 조달
- 세력끼리 통정매매: 100원 매도 → 100원 매수 → 101원 매도 → 101원 매수…
- 활발한 거래처럼 위장 (가짜 거래량 생성)

**2단계 — 정보 유포 (선수 투입)**
- 증권사 직원, 애널리스트, 경제 기자 등 "선수" 활용
- 커뮤니티, 방송, 메신저를 통한 가짜 or 과장 정보 유포
- 매집이 덜 됐으면 악재 유포 → 개미 팔게 만들기
- 매집 충분하면 호재 유포 → 개미 사게 만들기

**3단계 — 개미 유입 및 폭등**
- 개인 투자자들이 뉴스/정보 보고 매수
- 거래량 폭증 + 주가 폭등

**4단계 — 세력 탈출 (폭락)**
- 세력이 보유 물량 전량 매도
- 대량 매물 출하로 주가 폭락
- 가짜 호재 판명 → 원래 가치로 복귀
- 세력은 차익 실현, 개미는 고점 물림

#### 실제 사례
- 루보 (2006년)
- 3천리 (2022~2023년)
- 도이치모터스 (2010~2011년)

#### 리스크 신호 (플랫폼 경고 활용)
- 투자주의 → 투자경고 → 투자위험 (단계적 경고)
- 경고 신호 = 신중 검토 필요 (호재 급등도 뜰 수 있음)

## StockVision 적용 시사점

### 코드화 가능한 거래량 분석 규칙

#### 1. 거래량 MA 돌파 감지
```
volume_ma = MA(volume, 10)
volume_surge = volume > volume_ma * N  # N = 1.5~3.0 배수 임계값
```

#### 2. 거래량 소진 감지 (진입 대기 조건)
```
volume_drying = MA(volume, 5) < volume_ma * 0.5  # MA 50% 미만
price_in_range = low > support_line AND high < resistance_line
entry_ready = volume_drying AND price_in_range
```

#### 3. 패턴 분류 자동화
| 조건 | 패턴 | 신호 |
|------|------|------|
| volume↑ + price↑ | 패턴 A | 중립 (매물대 형성) |
| volume↑ + price↓ | 패턴 B | 즉시 매도 |
| volume↓ + price↑ | 패턴 C | 강한 매수 |
| volume↓ + price↓ | 패턴 D | 매도 경고 |
| volume↓ + price 횡보 | 패턴 E | 2차 상승 대기 |

#### 4. 수급 필터 (소형주 기준)
- 시총 1,000~2,000억 종목: 분봉 거래대금 ≥ 20억 필터
- 순간 체결량 임계값: ≥ 3,000만 원

#### 5. 작전주 위험도 스크리닝
- 시가총액 < 500억 + 평균 거래량 < 10만 주 = 고위험 플래그
- 거래량 급증(통정매매 패턴): 거래량은 크지만 주가 변동이 없는 경우
- 투자경고/위험 종목 = 자동 제외 또는 경고 표시

### 전략 엔진 연계 포인트
- **거래량 폭발 → 박스권 형성 → 소진 → 방향 돌파**를 이벤트 시퀀스로 정의
- 패럴렐 채널 자동 생성 (거래량 폭발 이후 N캔들 범위)
- 손익비 1:3 자동 TP/SL 계산
- 재료(뉴스) 기반 필터: 텔레그램/HTS 뉴스 연동 시 거래량 급증 종목과 교차 검증
