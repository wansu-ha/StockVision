## 추천 추가 전략 — Tier 1 (검증됨, 인기 높음)

### 1. MACD 골든크로스 전략

- **핵심 로직**:
  - 매수: MACD 라인이 시그널선을 상향 돌파 (골든크로스) + RSI 50 이상 (추세 확인)
  - 매도: MACD 라인이 시그널선을 하향 돌파 (데드크로스) OR 수익률 익절/손절
- **적합 장세**: 추세장 (상승/하락 추세가 명확한 시기)
- **검증 수준**: 수십 년 검증된 고전 지표. 한국 HTS(키움, MTS 등) 기본 전략으로 내장
- **난이도**: 초보
- **인기도**: 매우 높음 — 한국 주식 유튜브, 증권사 교육 자료에서 항상 1순위로 언급
- **DSL 예시**:
  ```
  -- MACD 골든크로스 + RSI 추세 확인 진입
  상향돌파(MACD(), MACD_SIGNAL()) AND RSI(14) >= 50 AND 보유수량 == 0 → 매수 100%

  -- 데드크로스 청산
  하향돌파(MACD(), MACD_SIGNAL()) AND 보유수량 > 0 → 매도 전량

  -- 고정 손절
  수익률 <= -3 → 매도 전량
  ```
- **출처/근거**: 키움증권 시스템트레이딩 가이드, CME Group 교육자료, 삼성증권 기술적 분석 강의

### 2. 볼린저밴드 + RSI 역추세 전략 (BB+RSI)

- **핵심 로직**:
  - 매수: 현재가가 볼린저밴드 하단 이하 + RSI 30 이하 (이중 과매도 확인)
  - 매도: 현재가가 볼린저밴드 상단 도달 OR RSI 70 이상 OR 손절
- **적합 장세**: 횡보장, 박스권 (추세 없는 구간에서 평균 회귀)
- **검증 수준**: 한국 파이썬 자동매매 입문 교재(브런치, WikiDocs)에서 "첫 번째 자동매매 알고리즘"으로 가장 많이 소개됨. 백테스트 결과 다수 공개
- **난이도**: 초보~중급
- **인기도**: 매우 높음 — "BB+RSI"로 검색하면 수십 개의 구현 예제와 유튜브 영상 존재
- **DSL 예시**:
  ```
  -- 볼린저밴드 하단 이탈 + RSI 과매도 동시 충족
  현재가 <= 볼린저_하단(20) AND RSI(14) <= 30 AND 보유수량 == 0 → 매수 100%

  -- 볼린저밴드 상단 도달 OR RSI 과매수
  현재가 >= 볼린저_상단(20) OR RSI(14) >= 70 → 매도 전량

  -- 손절
  수익률 <= -4 → 매도 전량
  ```
- **출처/근거**: brunch.co.kr/@ende/150 ("첫 번째 자동매매 알고리즘: BB+RSI"), KIS Code Assistant 예제, FMZ 전략 라이브러리

### 3. 스토캐스틱 과매도 반등 전략

- **핵심 로직**:
  - 매수: 스토캐스틱 %K가 %D를 과매도 구간(20 이하)에서 상향 돌파 (골든크로스) + MA 추세 확인
  - 매도: 스토캐스틱 80 이상 (과매수 구간) OR 손절/익절
- **적합 장세**: 횡보장, 단기 눌림목 반등
- **검증 수준**: 한국 단타 트레이더 사이에서 RSI와 함께 가장 많이 사용되는 오실레이터. 업비트 투자자보호센터, CME Group에서 기본 지표로 교육
- **난이도**: 중급 (스토캐스틱 파라미터 튜닝 필요)
- **인기도**: 높음 — RSI와 비교해서 더 민감한 신호를 원하는 트레이더 선호
- **DSL 예시**:

  > 현재 DSL v2에 `STOCH_K()`, `STOCH_D()` 함수 미구현 상태. 구현 후 작성 가능.
  ```
  -- (의사 코드 — STOCH 지표 추가 후 구현)
  -- 스토캐스틱 K가 D를 20 이하에서 상향 돌파 + MA 상승 추세
  상향돌파(STOCH_K(5,3), STOCH_D(5,3)) AND STOCH_K(5,3) <= 25 AND MA(20) > MA(60) AND 보유수량 == 0 → 매수 100%
  수익률 >= 4 OR STOCH_K(5,3) >= 80 → 매도 전량
  수익률 <= -3 → 매도 전량
  ```
- **출처/근거**: 업비트 투자자보호센터 스토캐스틱 교육, NANUM TRADING 지표 조합 가이드, 브런치 "스토캐스틱, 너의 정체는?"

### 4. MACD 히스토그램 다이버전스 전략

- **핵심 로직**:
  - 매수: 가격은 신저점인데 MACD 히스토그램은 상승 중 (강세 다이버전스) → 하락 모멘텀 소진 신호
  - 매도: 가격은 신고점인데 MACD 히스토그램이 하락 중 (약세 다이버전스) OR 수익률 익절
- **적합 장세**: 추세 전환점 포착, 추세 말기 진입 방지
- **검증 수준**: 전문 트레이더 선호 전략. 알파스퀘어, EBC Financial Group 등 기술적 분석 고급 교재에서 언급
- **난이도**: 고급 (다이버전스 패턴 인식 로직 복잡)
- **인기도**: 중간 — 입문자보다 중급 이상 트레이더 사용
- **DSL 예시**:

  > 다이버전스는 이전 N봉 저점 비교가 필요하여 현재 DSL v2 문법으로 직접 표현 어려움. `저점대비()` 같은 커스텀 필드 또는 히스토그램 턴 감지 함수 필요.
  ```
  -- (단순화 버전 — MACD 히스토그램이 음수에서 양수로 전환)
  상향돌파(MACD_HIST(), 0) AND RSI(14) < 50 AND 보유수량 == 0 → 매수 100%
  하향돌파(MACD_HIST(), 0) AND 보유수량 > 0 → 매도 전량
  수익률 <= -3 → 매도 전량
  ```
- **출처/근거**: alphasquare.co.kr MACD 기술적 지표 해설, EBC Financial Group 다이버전스 가이드, TradingView 커뮤니티

### 5. 이동평균 5/20/60일 삼선 정배열 전략

- **핵심 로직**:
  - 매수: MA(5) > MA(20) > MA(60) 정배열 달성 순간 진입 (추세 강도 확인)
  - 매도: MA(5) < MA(20) 역배열 전환 시 (추세 이탈) OR 손절
- **적합 장세**: 강한 상승 추세장
- **검증 수준**: 한국 주식 유튜브 채널 (토스증권 100% 활용법 등)에서 "필수 지표 조합"으로 항상 언급. 실전 투자자 커뮤니티(카페, 블라인드)에서 기본 세팅으로 공유
- **난이도**: 초보
- **인기도**: 매우 높음 — 국내 주식 차트 분석 1순위 패턴
- **DSL 예시**:
  ```
  -- 5/20/60 정배열 + RSI 과열 아닌 구간 진입
  정배열() = MA(5) > MA(20) AND MA(20) > MA(60)

  정배열() AND RSI(14) < 60 AND 보유수량 == 0 → 매수 100%

  -- 역배열 청산
  MA(5) < MA(20) AND 보유수량 > 0 → 매도 전량

  -- 손절
  수익률 <= -3 → 매도 전량
  ```
- **출처/근거**: 토스증권 차트 설정 가이드, 이동평균선 나무위키, EBC Financial Group 이동평균선 설명

### 6. 볼린저밴드 스퀴즈 + 돌파 전략

- **핵심 로직**:
  - 관찰: 볼린저밴드 폭이 좁아지는 구간 (변동성 수축 = 스퀴즈) 대기
  - 매수: 밴드 폭 확장과 함께 현재가가 볼린저 상단을 상향 돌파
  - 매도: 고점 대비 트레일링 OR 수익률 목표
- **적합 장세**: 횡보 후 추세 돌파 구간 (박스권 탈출 포착)
- **검증 수준**: XS.com 볼린저밴드 가이드, 알파스퀘어에서 "가장 수익성 높은 볼린저밴드 활용법"으로 소개
- **난이도**: 중급 (이격도 기반 스퀴즈 감지 필요)
- **인기도**: 높음 — 추세 돌파 매매자 선호
- **DSL 예시**:
  ```
  -- 볼린저밴드 상단 돌파 + 거래량 확인 (스퀴즈 돌파)
  상향돌파(현재가, 볼린저_상단(20)) AND 거래량 > 평균거래량(20) * 1.5 AND 보유수량 == 0 → 매수 100%

  -- 트레일링 청산
  고점 대비 <= -2 → 매도 전량

  -- 손절
  수익률 <= -3 → 매도 전량
  ```
- **출처/근거**: XS.com 볼린저밴드 완벽 가이드, alphasquare.co.kr 볼린저밴드 기술적 지표, learningspoons.com 볼린저밴드 매매법

### 7. 장 초반 급등 모멘텀 전략 (오전 매수)

- **핵심 로직**:
  - 매수: 장 시작 후 일정 시간 이내 + 등락률이 큰 양수 (갭 상승 또는 급등) + 거래량 폭발
  - 매도: 장마감 전 강제 청산 (당일매매) OR 수익률 목표/손절
- **적합 장세**: 테마주·이슈주 당일 모멘텀, 갭 상승 지속 종목
- **검증 수준**: 한국 단타 트레이더 커뮤니티(나무위키 단타매매기법, 블로그 다수)에서 오전 9시~10시 집중 매매 전략으로 검증
- **난이도**: 중급 (종목 필터링과 리스크 관리 중요)
- **인기도**: 높음 — 단타 투자자 사이 매우 인기, 자동화 시 갭 체이싱 위험 주의
- **DSL 예시**:
  ```
  -- 장 시작 후 30분 이내 + 등락률 3% 이상 + 거래량 폭발
  장시작후 <= 30 AND 등락률 >= 3 AND 거래량 > 평균거래량(20) * 3 AND 보유수량 == 0 → 매수 100%

  -- 수익 목표 익절
  수익률 >= 3 → 매도 전량

  -- 손절 (빠른 컷)
  수익률 <= -2 → 매도 전량

  -- 시간 청산 (오후 보유 방지)
  장시작후 >= 180 AND 보유수량 > 0 → 매도 전량
  ```
- **출처/근거**: 나무위키 단타매매기법, 브런치 단타매매 가이드, 키움 자동매매 커뮤니티 실전 전략

### 8. EMA 크로스 추세 추종 전략

- **핵심 로직**:
  - 매수: EMA(단기)가 EMA(장기)를 상향 돌파 (기존 MA 골든크로스와 유사하나 EMA는 최근 가격 반응 빠름)
  - 매도: EMA(단기)가 EMA(장기)를 하향 돌파 OR 손절
- **적합 장세**: 추세장 (MA 크로스 대비 신호 빠름, 노이즈 약간 더 많음)
- **검색 수준**: 트레이딩뷰 Pine Script 예제, 유튜브 "EMA 5일 20일 교차 매매" 콘텐츠 다수. 암호화폐 자동매매에서 특히 인기
- **난이도**: 초보
- **인기도**: 높음 — MA 크로스와 함께 자동매매 입문 전략의 양대산맥
- **DSL 예시**:
  ```
  -- EMA 12/26 골든크로스 진입
  상향돌파(EMA(12), EMA(26)) AND 보유수량 == 0 → 매수 100%

  -- EMA 데드크로스 청산
  하향돌파(EMA(12), EMA(26)) AND 보유수량 > 0 → 매도 전량

  -- 손절
  수익률 <= -3 → 매도 전량
  ```
- **출처/근거**: Threads @kjh__2988 EMA 5/20일 교차 매매법, EBC Financial Group 이동평균선 설정, TradingView EMA 크로스 전략

## Tier 2 (관심 있지만 1차에서 제외)

| 전략명 | 제외 이유 |
|--------|----------|
| **터틀 트레이딩** (20/55일 고가 돌파 매수) | 한국 단일 종목 적용 시 장기 보유 전제 필요. 포트폴리오 레벨 전략으로 단일 종목 엔진과 미스매치 |
| **퀀트 팩터 전략** (저PBR + 모멘텀 콤보) | 종목 스크리닝 단계 전략 — 매매 조건보다 종목 선정 로직. DSL보다 필터링 파이프라인에 적합 |
| **RSI 다이버전스** | MACD 다이버전스와 유사 원리. Tier 1의 MACD 히스토그램 전략으로 커버 가능 |
| **VWAP 이탈 전략** | 분봉 VWAP 필요 — 현재 DSL v2는 일봉 기준 지표만 지원 (v2 분봉 지원 후 재검토) |
| **파라볼릭 SAR** | 추세 추종 자체는 기존 MA 크로스 전략과 중복. 복잡도 대비 차별성 낮음 |
| **레인보우 이평선** (3/5/8/13/21/55 다중 MA) | 진입 조건이 복잡하여 DSL 표현 장황해짐. 실용 효과 대비 코드 복잡도 높음 |
| **캔들 패턴 (망치형, 도지 등)** | 캔들 패턴 감지 함수 DSL에 미구현. 구현 비용 대비 신뢰도 논란 있음 |

## 인사이트

### 공통 패턴

1. **지표 단독 사용 지양**: 한국 시스템매매 교육 자료 대부분이 "RSI 하나만 쓰지 마라, 볼린저밴드나 MACD와 함께 써라"를 강조. 이미 구현된 프리셋들도 이 패턴을 따름.

2. **거래량 확인 필수**: 국내 실전 트레이더 커뮤니티에서 "거래량 없는 신호는 페이크"가 공통 인식. 볼린저밴드 스퀴즈 돌파, 급등 모멘텀 전략 모두 거래량 조건 추가.

3. **손절 없는 전략은 없음**: 검색된 모든 실전 전략에서 손절(손실 제한) 조건을 필수로 포함. "손절이 가장 중요한 조건"이라는 언급 반복.

4. **진입보다 청산이 어렵다**: 한국 주식 자동매매 커뮤니티의 공통 교훈. 이미 구현된 프리셋(multi-exit, breakeven-trailing, atr-dynamic)이 청산 전략에 집중된 것과 일치.

### 추가 시 고려사항

- **STOCH 지표 추가**: Tier 1 전략 3번(스토캐스틱)을 온전히 구현하려면 `STOCH_K()`, `STOCH_D()` 함수를 DSL에 추가해야 함.
- **중복 최소화**: Tier 1 전략 1번(MACD 골든크로스)과 기존 `time-sequential` 프리셋은 골든크로스를 공통으로 사용하나, MACD 골든크로스는 다른 지표(MACD vs MA)라 차별성 있음.
- **난이도 분산**: 현재 7개 프리셋은 중급~고급 느낌. Tier 1의 1번, 5번, 8번은 초보 진입 장벽 낮춰주는 전략으로 적합.
