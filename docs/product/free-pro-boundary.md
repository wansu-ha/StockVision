## 1. 목적

이 문서는 현재 논의를 기준으로 StockVision의 무료 / Pro / Team 경계를 더 명확하게 정리한 문서다.
핵심 원칙은 다음과 같다.

- 무료는 `신뢰를 만드는 기본 경험`
- Pro는 `같은 비서를 어디서나 이어서 쓰는 경험`
- Team은 `여럿이 함께 운영하는 경험`

## 2. 한 줄 정의

- 무료: `내 PC에서 안전하게 쓰는 로컬 주식 비서`
- Pro: `어디서나 같은 비서를 이어서 쓰는 개인 주식 비서`
- Team: `전략과 기록을 함께 운영하는 협업 비서 플랫폼`

## 3. 무료에 포함하는 것이 자연스러운 것

무료는 `기본 가치 체험`이 가능해야 한다.
사용자가 "이 제품은 믿을 만하다"고 느끼는 핵심은 무료에서 막지 않는다.

### 포함

- 로컬 서버 설치 및 실행
- 브로커 연동
- 전략 DSL 작성 / 수정
- 기본 백테스트
- 전략 실행 엔진
- 킬스위치, 손실 제한, 중복 주문 방지
- 사용자 BYO LLM 연결
- 같은 PC에서의 비서 대화
- 기본 아침 브리핑
- 기본 설명 가능성
- 로컬 로그 / 복기
- 원격 상태 확인의 기본 기능
- 로컬 서버 관제의 기본 상태 확인
- 기기 교체 복구의 기본 기능
- 최근 30일 전략 이력 / 롤백
- 전략 import/export
- 로컬 메모리

### 이유

- 이걸 막으면 제품이 쪼잔해 보인다.
- 사용자가 신뢰와 편의의 핵심을 체험하지 못한다.
- 오픈소스/로컬 구조와 충돌한다.

## 4. Pro의 핵심 가치

Pro는 `더 좋은 AI`가 아니라 `더 좋은 운영 경험`을 판다.

### 핵심 가치 1. 어디서나 같은 비서를 이어서 사용

- 모바일 웹/앱에서 비서와 대화
- 다른 PC에서 설치 없이 로그인 후 비서 사용
- 같은 비서 설정, 말투, 최근 맥락 연속성 유지
- 로컬 서버가 온라인이면 실시간 응답, 오프라인이면 마지막 동기화 기준 응답

### 핵심 가치 2. 직접 운영하는 귀찮음을 줄여줌

- 원격 접속과 릴레이
- 동기화
- 원격 브리핑 확인
- 더 편한 설정 UX
- 원격 대화형 운영 경험

### 핵심 가치 3. 더 오래 남고 더 잘 이어짐

- 장기 전략 이력 / 장기 롤백
- 장기 메모리 / 장기 복기 기록
- 장기 리포트 보관
- 다기기 연속성

## 5. Team의 핵심 가치

Team은 개인 사용자의 연장이 아니라 `협업용 운영 레이어`다.

### 포함 가치

- 공유 전략 폴더
- 권한 분리
- 승인 플로우
- 감사 로그
- 공유 브리핑 / 리포트
- 팀 메모리 및 플레이북
- 좌석 관리
- 우선 지원

## 7. 텔레그램과의 관계

텔레그램은 막을 대상이 아니라 `무료 진입 채널`이다.

### 무료 텔레그램 사용

- 사용자가 직접 봇 세팅
- 기본 알림 / 간단 질의
- 로컬 서버 연동

### Pro가 이기는 포인트

- 봇 세팅 없이 로그인만으로 바로 사용
- 웹/앱에서 더 나은 대화 UX
- 전략/상태/설명/승인/로그가 한 흐름으로 이어짐
- 같은 비서의 맥락 연속성

즉, Pro의 핵심은 `텔레그램보다 똑똑함`이 아니라 `텔레그램보다 훨씬 편한 운영 경험`이다.

## 8. 무료 / 유료 경계에서 뺄 것

아래 항목은 유료 잠금 대상으로 두지 않는다.

- 기본 브로커 연결
- 기본 로컬 엔진 실행
- 핵심 안전장치
- 기본 설명 가능성
- 기본 아침 브리핑
- 기본 원격 상태 확인
- 기기 교체 복구 기본
- 사용자 자신의 LLM 연결

## 9. Pro 첫 유료 포인트 3개

현재 기준으로 가장 설득력 있는 Pro 핵심은 아래 세 가지다.

1. `모바일/다른 PC에서 같은 비서와 이어서 대화`
2. `장기 이력 / 장기 메모리 / 장기 복구`
3. `고급 브리핑 + 관리형 운영 컨텍스트`

## 10. 가격 감각

초기 가격 가설은 기존 문서와 동일하게 아래 범위를 유지한다.

- Pro: 월 12,900원 ~ 19,900원
- Team: 월 69,000원 ~ 149,000원

권장 시작값:

- Pro: 월 14,900원 또는 16,900원
- Team: 3석 포함 월 79,000원

## 11. 제품 문장 예시

### 무료

- `내 PC에서 안전하게 쓰는 로컬 주식 비서`
- `키는 로컬에 두고, 전략은 직접 실행하는 오픈소스 엔진`

### Pro

- `모바일과 웹에서도 같은 비서를 이어서 쓰세요`
- `설치 없이 어디서나 내 전략과 계좌 상태를 이어서 확인하세요`
- `같은 비서가 아침 브리핑부터 장중 운영까지 계속 이어집니다`

### Team

- `전략을 혼자 보지 말고 함께 운영하세요`
- `공유, 승인, 감사 로그까지 갖춘 협업 비서 플랫폼`

## 12. 결론

무료는 제품의 진짜 가치를 체험할 수 있어야 한다.
따라서 핵심 실행 구조, 기본 비서 경험, 기본 설명과 복구는 무료에 가깝다.

유료는 `막아둔 기능`이 아니라,
`같은 비서를 기기를 넘어 이어서 쓰고`,
`귀찮은 운영을 대신 맡기고`,
`기록과 맥락이 더 오래 유지되는 경험`에서 만들어야 한다.

## 13. 구현 방식 (2026-03-17 확정)

초기(무료 플랜만 존재)에는 `.env` 환경변수로 제한값을 관리한다.

```env
PLAN_MAX_STRATEGIES=3
PLAN_HISTORY_DAYS=30
```

서버 config에서 `settings.PLAN_MAX_STRATEGIES`로 참조.
Pro 결제 시스템 도입 시 DB `plans` 테이블 + 사용자별 `plan_id`로 전환한다.
