기존 프리셋 7개에 추가로 8개 전략을 보강하고, 이를 위해 DSL v2 내장 함수를 확장한다.

| 포함 | 제외 |
|------|------|
| 내장 함수 확장 (7종) | 다른 종목 데이터 참조 (`종목()`) |

## 2. 내장 함수 확장

## 3. 추가 프리셋 8개

### 3.1 MACD 골든크로스

- **카테고리**: 추세
- **난이도**: 초보
- **장세**: 추세장
```
상향돌파(MACD(), MACD_SIGNAL()) AND RSI(14) >= 50 AND 보유수량 == 0 → 매수 100%
하향돌파(MACD(), MACD_SIGNAL()) AND 보유수량 > 0 → 매도 전량
수익률 <= -3 → 매도 전량
```

### 3.2 볼린저밴드 + RSI 역추세

- **카테고리**: 역추세
- **난이도**: 초보~중급
- **장세**: 횡보장
```
현재가 <= 볼린저_하단(20) AND RSI(14) <= 30 AND 보유수량 == 0 → 매수 100%
현재가 >= 볼린저_상단(20) OR RSI(14) >= 70 → 매도 전량
수익률 <= -4 → 매도 전량
```

### 3.3 스토캐스틱 과매도 반등

- **카테고리**: 역추세
- **난이도**: 중급
- **장세**: 횡보/눌림목
```
상향돌파(STOCH_K(5,3), STOCH_D(5,3)) AND STOCH_K(5,3) <= 25 AND MA(20) > MA(60) AND 보유수량 == 0 → 매수 100%
수익률 >= 4 OR STOCH_K(5,3) >= 80 → 매도 전량
수익률 <= -3 → 매도 전량
```

### 3.4 MACD 히스토그램 다이버전스

- **카테고리**: 추세 전환
- **난이도**: 고급
- **장세**: 추세 말기
```
강세다이버전스(MACD_HIST(), 20) AND RSI(14) < 50 AND 보유수량 == 0 → 매수 100%
약세다이버전스(MACD_HIST(), 20) AND 보유수량 > 0 → 매도 전량
수익률 <= -3 → 매도 전량
```

### 3.5 삼선 정배열 (5/20/60)

- **카테고리**: 추세
- **난이도**: 초보
- **장세**: 강 상승장
```
MA(5) > MA(20) AND MA(20) > MA(60) AND RSI(14) < 60 AND 보유수량 == 0 → 매수 100%
MA(5) < MA(20) AND 보유수량 > 0 → 매도 전량
수익률 <= -3 → 매도 전량
```

### 3.6 볼린저밴드 스퀴즈 돌파

- **카테고리**: 돌파
- **난이도**: 중급
- **장세**: 박스권 탈출
```
상향돌파(현재가, 볼린저_상단(20)) AND 거래량 > 평균거래량(20) * 1.5 AND 보유수량 == 0 → 매수 100%
고점 대비 <= -2 → 매도 전량
수익률 <= -3 → 매도 전량
```

### 3.7 장 초반 급등 모멘텀

- **카테고리**: 모멘텀
- **난이도**: 중급
- **장세**: 테마/이슈주
```
장시작후 <= 30 AND 등락률 >= 3 AND 거래량 > 평균거래량(20) * 3 AND 보유수량 == 0 → 매수 100%
수익률 >= 3 → 매도 전량
수익률 <= -2 → 매도 전량
장시작후 >= 180 AND 보유수량 > 0 → 매도 전량
```

### 3.8 EMA 크로스 추세 추종

- **카테고리**: 추세
- **난이도**: 초보
- **장세**: 추세장
```
상향돌파(EMA(12), EMA(26)) AND 보유수량 == 0 → 매수 100%
하향돌파(EMA(12), EMA(26)) AND 보유수량 > 0 → 매도 전량
수익률 <= -3 → 매도 전량
```

## 4. 전략 상태 요약 표시

경고/차단 대신, 전략 카드에 **사실만 표시**:

```
┌─────────────────────────────┐
│ 골든크로스 전략              │
│ 매수 조건 2개 | 매도 조건 1개│
│ ✅ 손절 있음                │
└─────────────────────────────┘

┌─────────────────────────────┐
│ 모멘텀 전략                 │
│ 매수 조건 1개 | 매도 조건 0개│
│ 💀 손절 없음                │
└─────────────────────────────┘
```

- parse_v2 결과에서 매수/매도 규칙 수, 손절 유무를 자동 분석
- 경고/차단 없음 — 사실 표시만
- 문법 에러(parse_v2 실패)만 저장 차단

## 5. AI 코파일럿 연계

- AI가 전략 생성 시 시스템 프롬프트에서 **손절 항상 포함** 지시
- 프리셋 전체에 손절 포함 → 자연스러운 교육 효과
- 사용자가 직접 손절을 제거하면 → 상태 요약에 "손절 없음" 표시 (본인 판단 존중)
