# DSL v2 문법 레퍼런스

> 코드 기준 (`sv_core/parsing/builtins.py`, `parser.py`, `evaluator.py`) — 2026-03-29

## 기본 구조

```
-- 주석
상수이름 = 값
커스텀함수 = 조건식
조건 → 행동
조건 -> 행동        (ASCII 화살표도 가능)
```

## 상수 선언

```
손절 = -3
기간 = 14
tf = "5m"
```

숫자 또는 `"문자열"`. 규칙 안에서 이름으로 참조.

## 커스텀 함수 선언

```
과매도 = RSI(14) <= 30
과매도() = RSI(14) <= 30    (괄호 형태도 가능)
```

우변이 단순 숫자/문자열이 아니면 커스텀 함수로 인식. 규칙 조건에서 이름으로 호출.

## 규칙 (조건 → 행동)

```
RSI(14) < 30 → 매수 100%
수익률 >= 5 → 매도 전량
고점 대비 <= -5 → 매도 전량
RSI(14) > 70 → 매도 50%
```

### 행동 형식

| 형식 | 의미 |
|------|------|
| `매수 N%` | 가용 자금의 N% 매수 |
| `매도 전량` | 보유 전량 매도 |
| `매도 나머지` | `전량`과 동일 |
| `매도 N%` | 보유 수량의 N% 매도 |

### 우선순위 (여러 규칙 동시 충족 시)

1. 매도 전량 > 매도 N% > 매수
2. 같은 종류 내에서는 먼저 선언된 규칙 우선

## 내장 필드

괄호 없이 바로 사용. 모두 숫자 타입.

### 시세 필드

| 필드 | 설명 |
|------|------|
| `현재가` | 현재 주가 |
| `거래량` | 현재 거래량 |

### 포지션 필드

| 필드 | 설명 |
|------|------|
| `수익률` | 현재 수익률 (%) |
| `보유수량` | 보유 주식 수 |
| `고점 대비` | 보유 중 최고가 대비 현재 하락률 (%) |
| `수익률고점` | 보유 중 달성한 최고 수익률 (%) |
| `진입가` | 평균 진입 단가 |
| `보유일` | 영업일 기준 보유 일수 |
| `보유봉` | 평가 사이클 기준 보유 봉 수 |

### 시간 필드

| 필드 | 설명 |
|------|------|
| `시간` | 현재 시각 HHMM (정수, 예: 0930) |
| `장시작후` | 장 시작(09:00) 후 경과 분 |
| `요일` | 1=월 ~ 5=금 |

### 상태 필드

| 필드 | 설명 |
|------|------|
| `실행횟수` | 현재 포지션 내 이 규칙의 실행 횟수 |

## 내장 함수

괄호 필수. 마지막 인자로 타임프레임(`"5m"`, `"1d"` 등)을 선택적으로 전달 가능.

### 지표 함수 (반환: 숫자)

| 함수 | 인자 | 예시 |
|------|------|------|
| `RSI(기간)` | 1~2 | `RSI(14)`, `RSI(14, "5m")` |
| `MA(기간)` | 1~2 | `MA(20)` |
| `EMA(기간)` | 1~2 | `EMA(12)` |
| `MACD()` | 0~1 | `MACD()`, `MACD("5m")` |
| `MACD_SIGNAL()` | 0~1 | `MACD_SIGNAL()` |
| `볼린저_상단(기간)` | 1~2 | `볼린저_상단(20)` |
| `볼린저_하단(기간)` | 1~2 | `볼린저_하단(20)` |
| `평균거래량(기간)` | 1~2 | `평균거래량(20)` |
| `ATR(기간)` | 1~2 | `ATR(14)` |
| `최고가(기간)` | 1~2 | `최고가(20)` |
| `최저가(기간)` | 1~2 | `최저가(20)` |
| `이격도(기간)` | 1~2 | `이격도(20)` |

### 크로스오버 함수 (반환: boolean)

| 함수 | 인자 | 설명 |
|------|------|------|
| `상향돌파(A, B)` | 2 | 이전: A < B, 현재: A >= B |
| `하향돌파(A, B)` | 2 | 이전: A > B, 현재: A <= B |

### 상태 함수 (반환: 숫자)

| 함수 | 인자 | 설명 |
|------|------|------|
| `횟수(조건, 기간)` | 2 | 최근 N사이클 중 조건이 True인 횟수 |
| `연속(조건)` | 1 | 현재 연속 True 봉 수 (False 시 리셋) |

## 내장 패턴 함수

괄호 없이 사용. boolean 반환. 내부적으로 DSL 표현식으로 전개.

| 이름 | 전개 |
|------|------|
| `골든크로스` | `상향돌파(MA(5), MA(20))` |
| `데드크로스` | `하향돌파(MA(5), MA(20))` |
| `RSI과매도` | `RSI(14) <= 30` |
| `RSI과매수` | `RSI(14) >= 70` |
| `볼린저하단돌파` | `현재가 <= 볼린저_하단(20)` |
| `볼린저상단돌파` | `현재가 >= 볼린저_상단(20)` |
| `MACD골든크로스` | `상향돌파(MACD(), MACD_SIGNAL())` |
| `MACD데드크로스` | `하향돌파(MACD(), MACD_SIGNAL())` |

## 연산자

| 연산자 | 우선순위 | 설명 |
|--------|---------|------|
| `OR` | 1 (낮음) | 논리합 |
| `AND` | 2 | 논리곱 |
| `NOT` | 3 | 부정 |
| `>`, `>=`, `<`, `<=`, `==`, `!=` | 4 | 비교 (체이닝 불가) |
| `BETWEEN low AND high` | 4 | `>= low AND <= high`로 전개 |
| `+`, `-` | 5 | 덧셈/뺄셈 |
| `*`, `/` | 6 | 곱셈/나눗셈 |
| `-` (단항) | 7 | 부호 반전 |

## Index Access

```
현재가[1]       -- 1봉 전 현재가
RSI(14)[3]     -- 3봉 전 RSI(14)
MA(20)[0]      -- 현재 MA(20) (= MA(20))
```

범위: 0~60. 정수만 허용.

## v1 호환

`매수:`/`매도:` 블록도 v2 파서가 자동 변환.

```
매수: RSI(14) < 30    →  RSI(14) < 30 AND 보유수량 == 0 → 매수 100%
매도: 수익률 >= 5     →  수익률 >= 5 → 매도 전량
```

## 예시: 복합 전략

```
-- 상수
손절 = -3
익절 = 10
추매기준 = -5

-- 커스텀 함수
과매도 = RSI(14) <= 30

-- 규칙
과매도 AND 보유수량 == 0 → 매수 100%
수익률 <= 추매기준 AND 실행횟수 < 2 → 매수 50%
수익률 >= 익절 AND 고점 대비 <= 손절 → 매도 전량
수익률 <= 손절 * 3 → 매도 전량
```
