## [0.3.0] - 2025-08-22

### 추가됨
- **프론트엔드 UI/UX 대폭 개선**
  - 대시보드 완전 재설계: 현대적이고 직관적인 카드 기반 레이아웃
  - HeroUI 라이브러리 도입: React 19 호환 UI 컴포넌트 시스템
  - 반응형 디자인: 모바일 우선 접근법으로 모든 디바이스 최적화
  - 그라디언트 및 그림자: 시각적 깊이감과 현대적인 디자인 요소

- **기술적 개선사항**
  - Tailwind CSS v3 최적화: PostCSS 설정 및 플러그인 구성 개선
  - HeroUI 통합: Tailwind 플러그인과 테마 시스템 완벽 연동
  - Vite 설정 개선: PostCSS 설정 파일 분리 및 최적화
  - 모듈 시스템 통일: ES Modules 기반으로 모든 설정 파일 통일

- **컴포넌트 개선**
  - Dashboard.tsx: HeroUI Card, CardHeader, CardBody 컴포넌트 적용
    - 시장 개요 카드 4개 (총 시가총액, 등록된 주식, AI 예측 정확도, 가상 거래)
    - 최근 등록된 주식 테이블
    - 빠른 액션 카드 3개 (AI 예측, 가상 거래, 백테스팅)
  - Layout.tsx: 네비게이션 및 레이아웃 구조 개선
  - StockList.tsx: 주식 목록 표시 개선

- **사용자 경험 개선**
  - 로딩 상태: AI 데이터 분석 중 애니메이션 표시
  - 에러 처리: 백엔드 연결 실패 시 명확한 안내 메시지
  - 시각적 피드백: 호버 효과, 트랜지션, 그림자 효과
  - 정보 구조화: 카드 기반으로 정보를 논리적으로 그룹화

- **성능 최적화**
  - CSS 번들 최적화: 불필요한 CSS 제거 및 HeroUI CSS-in-JS 활용
  - 컴포넌트 지연 로딩: React.lazy를 통한 코드 스플리팅 준비
  - 이미지 최적화: SVG 아이콘 및 최적화된 이미지 사용

- **문제 해결**
  - CSS 적용 문제: PostCSS 설정 및 Tailwind 플러그인 문제 해결
  - React 19 호환성: 최신 React 버전과 호환되는 UI 라이브러리 선택
  - 모듈 충돌: require vs import 문제 해결 및 ES Modules 통일

- **데이터 연동**
  - 실시간 주식 데이터: yfinance를 통한 1년치 실제 시장 데이터 (249개 거래일)
  - API 연동: 백엔드와의 완벽한 데이터 연동 및 에러 처리
  - 동적 데이터 표시: 실제 주식 정보를 기반으로 한 대시보드 구성

## [0.2.0] - 2025-08-22

### 추가됨

  - Vite 기반 현대적인 React 애플리케이션

  - Tailwind CSS를 활용한 반응형 디자인

- **차트 시각화 컴포넌트**
  - Recharts 기반 주식 가격 차트 (OHLCV)
  - 거래량 차트 컴포넌트
  - 한국어 로컬라이제이션 지원

- **상태 관리 및 데이터 페칭**

- **UI/UX 컴포넌트**
  - Layout 컴포넌트 - 네비게이션 및 헤더
  - 로딩 상태 및 오류 처리

## [0.1.0] - 2025-08-21

### 추가됨
- **프로젝트 초기화**
  - StockVision 프로젝트 구조 생성
  - Git 저장소 초기화
  - 포괄적인 .gitignore 파일 추가
  - 프로젝트 문서화 구조 생성

- **백엔드 인프라**
  - CORS 미들웨어가 포함된 FastAPI 애플리케이션 설정
  - Python 가상 환경 (Python 3.13.7)
  - 의존성 관리 (requirements.txt)
  - 프로젝트 구조: `app/{api,core,models,services,utils}`

- **데이터베이스 설계**
  - SQLite 데이터베이스 연결 (개발용)
  - 주식, 가격, 기술적 지표용 데이터베이스 모델
  - 가상 거래 시스템용 데이터베이스 모델
  - 자동 거래 및 백테스팅용 데이터베이스 모델
  - 데이터베이스 초기화 스크립트

- **데이터 수집 시스템**
  - 주식 데이터 수집을 위한 yfinance 통합
  - 주식 정보 수집 (심볼, 회사명, 섹터, 산업, 시가총액)
  - 과거 가격 데이터 수집 (OHLCV)
  - 데이터 검증 및 오류 처리
  - 중복 방지가 포함된 데이터베이스 저장

- **기술적 지표 서비스**
  - RSI (상대강도지수) 계산
  - EMA (지수이동평균) - 20일 및 50일
  - MACD (이동평균수렴확산) 계산
  - 볼린저 밴드 계산
  - Pandas 기반 효율적인 계산
  - 계산된 지표의 데이터베이스 저장

- **RESTful API 구현**
  - `GET /stocks/` - 모든 주식 목록
  - `GET /stocks/{symbol}` - 주식 상세 정보
  - `GET /stocks/{symbol}/prices` - 날짜 범위별 가격 데이터
  - `GET /stocks/{symbol}/indicators` - 기술적 지표
  - `GET /stocks/{symbol}/summary` - 최신 데이터가 포함된 주식 요약
  - 적절한 오류 처리 및 HTTP 상태 코드
  - 쿼리 매개변수 검증

- **데이터 모델**
  - 기본 정보가 포함된 Stock 모델
  - OHLCV 데이터용 StockPrice 모델
  - 계산된 지표용 TechnicalIndicator 모델
  - VirtualAccount, VirtualTrade, VirtualPosition 모델
  - AutoTradingRule 및 BacktestResult 모델
  - 성능 최적화를 위한 적절한 인덱싱

- **서비스 계층**
  - 주식 데이터 수집용 DataCollector 서비스
  - 지표 계산용 TechnicalIndicatorCalculator 서비스
  - 데이터베이스 세션 관리
  - 오류 처리 및 로깅

- **테스트 및 검증**
  - 데이터 수집 테스트 스크립트
  - 기술적 지표 계산 테스트
  - curl 명령어를 사용한 API 엔드포인트 테스트
  - 5개 주요 주식 데이터 성공적 수집

### 기술적 세부사항
- **수집된 데이터**: 5개 주식 (AAPL, MSFT, GOOGL, AMZN, TSLA)
- **가격 데이터**: 주식당 249개 일일 OHLCV 기록 (총 1,245개)
- **기술적 지표**: 주식당 2,171개 지표 (총 10,855개)
- **데이터 기간**: 2024년 8월 - 2025년 1월
- **데이터베이스**: 적절한 인덱싱이 포함된 SQLite
- **API 성능**: 적절한 오류 처리가 포함된 빠른 응답 시간
